# 융합 분석 방법론

## 1. 갭 분석 (Gap Analysis)

두 점수 사이의 차이에서 의미를 읽는다.

```
갭 공식:
갭 = |점수A - 점수B|

해석 기준:
- 40점 이상: 극단적 갭 → 정체성의 핵심, 또는 방어기제
- 30-40점: 강한 갭 → 명확한 우선순위, 의도적 선택
- 20-30점: 유의미한 갭 → 상황에 따라 조절 가능
- 20점 미만: 약한 갭 → 유동적, 맥락 의존적
```

### 핵심 갭 패턴

**Mastery-Connection 갭** (깊이 vs 관계)
- 갭 30+ → "고독한 전문가" 패턴
- 혼자 깊이 파는 것을 압도적으로 선호
- 리스크: 고립, 지식의 사장, 후계자 부재, 인정 결핍

**Freedom-Security 갭** (자유 vs 안정)
- 갭 40+ → "속박 트라우마" 또는 "정착 공포" 가능성
- 둘 다 극단(하나 75+, 하나 35-)이면 방어기제 의심
- 과거에 갇혀본 경험, 또는 안정이 무너진 경험 탐색

**Achievement-Recognition 관계**
- 보통: Achievement ≥ Recognition
- 역전 시: 승리보다 인정이 중요한 타입
- "이겨도 아무도 안 알아주면 허무"

**1위-8위 동기 갭**
- 갭 50+ → 극단적으로 선명한 정체성
- 갭 30- → 다면적, 상황에 따라 달라지는 정체성

## 2. 조합 분석 (Combination Analysis)

특정 점수들이 함께 나타날 때 만드는 패턴

```
패턴 1: 자율적 심층 몰입자
조건: Mastery 75+ AND Complexity 75+ AND Autonomy 75+
의미: 혼자 복잡한 문제에 깊이 파고드는 것이 최적 환경
독약: 팀 조율, 단순 반복, 마이크로매니징

패턴 2: 압박 성과자
조건: Achievement 75+ AND Competition 70+ AND Deadline 70+
의미: 경쟁과 마감이 있어야 엔진 점화
독약: 여유로운 환경, "천천히 해도 돼"

패턴 3: 관계 기반 안정 추구자
조건: Connection 70+ AND Security 70+ AND Audience 60+
의미: 팀의 일원으로서 인정받을 때 최적
독약: 혼자 던져놓기, 불확실한 환경

패턴 4: 영원한 탐험가
조건: Freedom 75+ AND Adventure 75+ AND Security 40-
의미: 정착 = 죽음, 항상 새로운 것을 찾아야 함
리스크: 아무것도 깊이 쌓이지 않음

패턴 5: 인정받고 싶은 고독한 전문가
조건: Mastery 80+ AND Recognition(숨겨진) 60+ AND Connection 55-
의미: 혼자 파고들지만, 아무도 안 알아주면 허무
위험: 가장 위험한 조합 - 인정 욕구 부정하면서 고립
```

## 3. 모순 탐지 (Contradiction Detection)

보통 함께 움직이는 쌍이 불일치할 때 특별한 의미

```
일반적 상관관계:
- Achievement ↔ Competition (보통 함께 높거나 낮음)
- Mastery ↔ Creation (지식 추구 ↔ 창작)
- Freedom ↔ Adventure (자유 ↔ 새로움)
- Connection ↔ Security (관계 ↔ 안정)
- Recognition ↔ Audience (인정 ↔ 주목)

모순 발생 시:
Achievement 고 + Competition 저 → 성과는 원하지만 경쟁은 싫어 (협력적 성취자? 트라우마?)
Freedom 고 + Adventure 저 → 자유는 원하지만 새로운 건 싫어 (익숙한 것 안에서의 자유)
Recognition 고 + Audience 저 → 인정받고 싶지만 주목받긴 싫어 (조용히 인정받고 싶음)
```

## 4. 동력 구조 융합 (Power Structure Fusion)

모든 분석을 하나의 동적 시스템으로 통합

```
융합 공식:

[점화조건 상위 2개] 
+ [에너지 패턴 (충전/소진)]
+ [숨겨진 동기]
+ [핵심 갈등]
+ [메타인지 수준]

= 이 사람의 "동력 구조"

예시:
점화: Complexity 88 + Autonomy 82
에너지: 과몰입 → 급격한 번아웃
숨겨진: Recognition
갈등: Mastery vs Adventure (48점 긴장)
메타인지: 70점

= 동력 구조:
"복잡한 문제에 자율적으로 파고들 때 점화됨.
그러나 멈추는 법을 모르고(메타인지 70으로 인식은 하지만 조절 못함),
인정받지 못하면 허무해짐.
깊이와 넓이 사이에서 에너지가 새어나감.

경로: 과몰입 → 인정 결핍 → 갈등 폭발 → 번아웃"
```

## 5. 시간축 예측 (Timeline Projection)

현재 패턴이 지속되면 어떻게 되는가

```
6개월 예측:
- 에너지 패턴 + 갈등 강도 + 번아웃 리스크로 계산
- 메타인지 70+ → 스스로 조절 가능성 있음
- 메타인지 50- → 외부 개입 없이는 패턴 반복

위기 예측:
- 갈등 강도 50+ → 6개월 내 내적 위기 가능성
- 번아웃 리스크 70+ → 3개월 내 경고 신호 예상
- 숨겨진 동기 결핍 지속 → 6개월 후 동기 저하

장기 예측 (10년):
- 현재 패턴이 10년 누적되면?
- 강점이 독이 되는 시점은?
- 지금 바꾸지 않으면?
```

### 15.3 취약점 구조

```
형식:
1. 급성 독약: [즉시 동력을 꺼버리는 것]

과몰입 (스스로 멈추지 못함)

붕괴 신호:
- '요즘 재미없다' = Recognition 결핍 시작

- 새 프로젝트 거부 = 이미 늦음"
```

```
혼자 일한 결과를 보여주고 싶은 것입니다.

해결책:

'나눔'이 아니라 '전시'입니다.

```

```

- 공급 주기: 2주마다 새로운 도전

2차 연료 (숨겨진): 성과 인정
- 주 1회 성과 공유 기회 또는 1:1에서 명시적 인정
- '이거 네가 한 거 대단하다'를 말로
- 주의: 본인이 '인정 필요 없다'고 해도 무시하세요. 필요합니다.
- 공급 주기: 주 1회 (결핍 시 6개월 후 동기 급감)"
```

```

- 새 프로젝트 제안에 시큰둥 = 3주 내 위기"
```

### 16.6 경고 신호

