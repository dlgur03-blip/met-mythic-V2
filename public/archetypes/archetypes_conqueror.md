## 원형 개요

### 핵심 정체성

**The Conqueror (정복자)**는 **목표 달성**에서 존재 의미를 찾는 원형입니다.

```
"멈추면 죽는다. 전진만이 답이다."
```

### 동기 가중치

| 동기 | 가중치 | 역할 |
|------|--------|------|
| **Achievement** | 0.70 | 핵심 동기 |
| **Recognition** | 0.20 | 보조 동기 |
| **Freedom** | 0.10 | 보조 동기 |

### 원형 일치 조건

```typescript
conditions: {
  achievement >= 70,
  achievement > mastery,
  achievement - security >= 20
}
```

### 4단계 성숙도 구조

| 레벨 | 이름 | 핵심 | 신화적 단계 |
|------|------|------|-------------|
| **Lv1** | 그림자 (Shadow) | 맹목적 파괴, 승리 집착 | 추락 전/암흑기 |
| **Lv2** | 각성 (Awakening) | 목표 인식, 방향 설정 | 여정의 시작 |
| **Lv3** | 통합 (Integration) | 전략적 승리, 책임 수용 | 성취의 정점 |
| **Lv4** | 초월 (Transcendence) | 승패를 넘어선 지혜 | 완성/승천 |

## 신화 인물 1: 아레스 (Ares)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | Ares (아레스) |
| **신화 체계** | 그리스 |
| **역할** | 전쟁의 신, 폭력과 파괴의 화신 |
| **상징** | 창, 투구, 독수리, 개, 멧돼지 |

### 동기 프로파일

```
Achievement: 0.95  ████████████████████ (핵심)
Freedom:     0.60  ████████████
Recognition: 0.50  ██████████
Security:    0.20  ████
Connection:  0.15  ███
Mastery:     0.30  ██████
```

### 핵심 특징 3가지

1. **순수한 전투 본능** — 승리 자체가 목적, 이유 불필요
2. **통제 불가능한 격정** — 한 번 점화되면 멈출 수 없음
3. **명예보다 승리** — 방법을 가리지 않음, 결과만 중요

### Lv1: 그림자 (Shadow) — "맹목적 파괴자"

#### 특징

```
"이기면 장땡이다. 뭘 부수든 상관없어."
```

- **행동 패턴**: 무분별한 경쟁, 사소한 것에도 승부욕 폭발
- **대인 관계**: 모든 관계를 경쟁으로 인식, 협력 불가
- **에너지 상태**: 폭발적이지만 지속 불가, 번아웃 반복
- **자기 인식**: "나는 이겨야만 가치 있다"

#### 신화적 대응

> **트로이 전쟁에서의 아레스**
> 
> 아레스는 트로이 전쟁에서 양측을 오가며 싸웠습니다.
> 누구 편이냐가 아니라, 싸움 자체가 목적이었기에.
> 아테나에게 패배했을 때, 제우스는 말했습니다:
> "너는 내게 가장 미운 신이다. 다툼만 좋아하니."
>
> **Lv1의 아레스는 싸움 자체에 중독된 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 승리 집착 | 지는 것을 견디지 못함, 사소한 패배에도 폭발 |
| 관계 파괴 | 동료도 경쟁 상대로 인식, 협력 관계 불가 |
| 자기 파괴 | 이기기 위해 자신까지 태움, 건강 무시 |
| 의미 상실 | 왜 이기려 하는지 잊음, 승리해도 공허함 |

#### 싱크로 신호 (이런 모습이면 Lv1)

- [ ] 회의에서 항상 "내가 맞다"고 주장해야 함
- [ ] 동료의 성공을 진심으로 축하할 수 없음
- [ ] 게임에서 져도 화가 남
- [ ] "왜 이기려 하는가?"라는 질문에 답할 수 없음
- [ ] 이긴 후에도 만족감이 없음

### Lv2: 각성 (Awakening) — "방향을 찾은 전사"

#### 특징

```
"이기고 싶다. 하지만 이제 뭘 위해 이기는지 안다."
```

- **행동 패턴**: 목표를 설정하고 경쟁, 무분별함 감소
- **대인 관계**: 경쟁과 협력 구분 시작, 팀 플레이 가능
- **에너지 상태**: 목표 있을 때 집중, 없을 때 방황
- **자기 인식**: "승리는 수단이다. 목적이 필요하다"

#### 신화적 대응

> **올림포스에서 인정받으려는 아레스**
>
> 아레스는 다른 신들에게 인정받고 싶었습니다.
> 하지만 그의 방식(폭력)은 올림포스에서 환영받지 못했습니다.
> 그는 자신의 힘이 "무엇을 위한 것인가"를 고민하기 시작합니다.
>
> **Lv2의 아레스는 방향을 찾아 헤매는 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 목표 의존 | 목표 없으면 무기력, 스스로 목표 설정 어려움 |
| 수단 고착 | 여전히 "이기는 것"만 방법으로 앎 |
| 인정 욕구 | 이기고 나서 인정받아야 만족 |
| 유연성 부족 | 패배를 배움으로 전환 못함 |

#### 싱크로 신호 (이런 모습이면 Lv2)

- [ ] 목표가 있으면 집중력 폭발, 없으면 무기력
- [ ] "왜 이기려 하는가?"에 대답할 수 있음
- [ ] 협력은 가능하지만 여전히 불편함
- [ ] 패배하면 분석보다 분노가 먼저
- [ ] 승리 후 인정받으면 만족, 못 받으면 불만

### Lv3: 통합 (Integration) — "전략적 정복자"

#### 특징

```
"이기되, 의미 있게 이긴다. 지더라도 배운다."
```

- **행동 패턴**: 전략적 경쟁, 승리의 질을 따짐
- **대인 관계**: 경쟁자를 존중, 좋은 라이벌 관계 형성
- **에너지 상태**: 지속 가능한 고성능, 회복 능력 보유
- **자기 인식**: "승리는 성장의 증거다"

#### 신화적 대응

> **아프로디테와의 사랑**
>
> 아레스는 아프로디테(미와 사랑의 여신)와 사랑에 빠집니다.
> 전쟁의 신이 사랑을 알게 된 것입니다.
> 그는 "파괴만이 전부가 아님"을 배웁니다.
> 승리 외에도 지킬 것이 생겼습니다.
>
> **Lv3의 아레스는 싸우는 이유를 찾은 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 과잉 책임 | 모든 싸움을 자기 싸움으로 만듦 |
| 완벽주의 | "의미 있는 승리"에 집착 |
| 지침 | 쉬지 않고 다음 목표로, 번아웃 위험 |

#### 싱크로 신호 (이런 모습이면 Lv3)

- [ ] 패배해도 배움을 찾음
- [ ] 경쟁자를 진심으로 존경할 수 있음
- [ ] "왜 이기려 하는가?"에 깊은 대답 가능
- [ ] 팀의 승리를 자기 승리만큼 기뻐함
- [ ] 승리 후 "다음은?"보다 "잘했다" 먼저

---

### Lv4: 초월 (Transcendence) — "전쟁을 넘어선 신"

#### 특징

```
"이김과 짐을 넘어, 싸움 자체가 수행이다."
```

- **행동 패턴**: 필요한 싸움만 선택, 불필요한 경쟁 거부
- **대인 관계**: 적도 스승으로 봄, 경쟁 자체가 연결
- **에너지 상태**: 고요한 힘, 폭발 없이도 강함
- **자기 인식**: "나는 전사다. 승패와 무관하게"

#### 신화적 대응

> **아레스의 드문 지혜**
>
> 신화에서 아레스가 Lv4에 도달한 장면은 드뭅니다.
> 하지만 그가 아프로디테와의 사이에서 낳은 아들 중
> "에로스(사랑)"와 "하르모니아(조화)"가 있다는 것은
> 전쟁의 신 안에도 초월의 가능성이 있음을 보여줍니다.
>
> **Lv4의 아레스는 파괴 속에서 창조를 보는 상태입니다.**

#### 도달 경지

| 경지 | 발현 |
|------|------|
| 선택적 전투 | 모든 싸움에 뛰어들지 않음, 의미 있는 것만 |
| 적의 성장 | 상대를 꺾되 성장시킴, 파괴가 아닌 단련 |
| 평화로운 힘 | 싸우지 않고도 강함을 발휘 |
| 존재 증명 불필요 | 승리로 자신을 증명할 필요 없음 |

#### 싱크로 신호 (이런 모습이면 Lv4)

- [ ] 이길 수 있지만 싸우지 않기를 선택할 수 있음
- [ ] 패배해도 자존감에 영향 없음
- [ ] 경쟁자의 성공을 돕고도 기쁨
- [ ] "이기는 것"보다 "성장하는 것"에 집중
- [ ] 싸움 없이도 평화로움

---

### 레벨 진화 조건

```
Lv1 → Lv2: "왜 이기려 하는가?"에 대답할 수 있을 때
Lv2 → Lv3: 패배에서 배움을 찾을 수 있을 때  
Lv3 → Lv4: 이기지 않고도 괜찮을 때
```

### 레벨 퇴화 경고

```
Lv4 → Lv3: 승리에 다시 집착이 생길 때
Lv3 → Lv2: 패배를 받아들이지 못할 때
Lv2 → Lv1: 목표를 잃고 맹목적 경쟁에 빠질 때
```

## 신화 인물 2: 토르 (Thor)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | Thor (토르) |
| **신화 체계** | 북유럽 |
| **역할** | 천둥의 신, 인간의 수호자 |
| **상징** | 묠니르(망치), 번개, 염소 |

### 동기 프로파일

```
Achievement: 0.85  █████████████████ (핵심)
Connection:  0.60  ████████████
Security:    0.50  ██████████
Recognition: 0.40  ████████
Freedom:     0.35  ███████
Mastery:     0.45  █████████
```

### 핵심 특징 3가지

1. **보호를 위한 힘** — 파괴가 아닌 수호가 목적
2. **단순하고 직접적** — 복잡한 계략보다 정면 돌파
3. **충성과 의리** — 동료를 위해 목숨을 건다

---

### Lv1: 그림자 (Shadow) — "무모한 힘자랑"

#### 특징

```
"내가 제일 세다. 증명해 보이겠다."
```

- **행동 패턴**: 힘으로 모든 것을 해결하려 함
- **대인 관계**: 약자를 무시, 강자에게 도전
- **에너지 상태**: 폭발적, 통제 불능
- **자기 인식**: "힘이 곧 나다"

#### 신화적 대응

> **거인 스크리미르와의 만남**
>
> 토르는 거인 스크리미르의 도전을 모두 실패합니다.
> 고양이 들어올리기(세계뱀), 노파와 레슬링(노쇠함), 뿔잔 비우기(바다).
> 그는 자신의 힘에만 의존했고, 지혜가 없었습니다.
>
> **Lv1의 토르는 힘만 믿고 무모하게 덤비는 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 힘 과신 | 모든 문제를 힘으로 해결하려 함 |
| 무모함 | 결과를 생각하지 않고 돌진 |
| 인정 갈망 | "최강"이라는 타이틀에 집착 |
| 단순 사고 | 복잡한 상황에서 판단 오류 |

#### 싱크로 신호 (이런 모습이면 Lv1)

- [ ] 문제가 생기면 "힘으로 밀어붙이면 된다"
- [ ] 조언을 받아들이지 못함
- [ ] 실패하면 "더 세게"가 해답이라 생각
- [ ] 약해 보이는 것을 극도로 싫어함
- [ ] 자신보다 강한 존재에게 무모하게 도전

---

### Lv2: 각성 (Awakening) — "지킬 것을 찾은 전사"

#### 특징

```
"내 힘은 무엇을 위한 것인가?"
```

- **행동 패턴**: 보호해야 할 대상 인식, 힘의 방향 설정
- **대인 관계**: 동료 개념 형성, 협력 시작
- **에너지 상태**: 목적 있을 때 안정적
- **자기 인식**: "힘은 도구다. 목적이 필요하다"

#### 신화적 대응

> **아스가르드의 수호자**
>
> 토르는 단순한 전사에서 "인간계(미드가르드)의 수호자"가 됩니다.
> 그의 힘은 이제 과시가 아닌 보호를 위한 것입니다.
> 거인들의 침략에서 인간을 지킵니다.
>
> **Lv2의 토르는 힘의 목적을 찾은 상태입니다.**

#### 싱크로 신호 (이런 모습이면 Lv2)

- [ ] "왜 이기려 하는가?"에 "지키기 위해"라고 답함
- [ ] 동료를 위해 힘을 사용함
- [ ] 무모함이 줄고 신중함 증가
- [ ] 그래도 여전히 힘에 의존하는 경향
- [ ] 지키지 못했을 때 자책이 큼

---

### Lv3: 통합 (Integration) — "현명한 천둥"

#### 특징

```
"싸울 때와 물러설 때를 안다."
```

- **행동 패턴**: 전략적 판단, 필요시 후퇴도 선택
- **대인 관계**: 로키 같은 복잡한 존재와도 협력
- **에너지 상태**: 지속 가능한 힘, 회복력 보유
- **자기 인식**: "힘은 책임이다"

#### 신화적 대응

> **로키와의 복잡한 관계**
>
> 토르는 로키의 배신에도 불구하고 그와 함께합니다.
> 단순히 "적/아군"으로 나누지 않습니다.
> 복잡한 관계를 받아들이고, 필요할 때 협력합니다.
>
> **Lv3의 토르는 힘과 지혜를 함께 사용하는 상태입니다.**

#### 싱크로 신호 (이런 모습이면 Lv3)

- [ ] 힘으로 해결할 수 없는 문제를 인정함
- [ ] 패배해도 "다음에 더 잘하면 된다"
- [ ] 복잡한 인간관계를 유지할 수 있음
- [ ] 후퇴가 필요할 때 후퇴함
- [ ] 약한 모습을 보여도 괜찮음

---

### Lv4: 초월 (Transcendence) — "라그나로크를 받아들인 신"

#### 특징

```
"나의 죽음조차 지키는 것의 일부다."
```

- **행동 패턴**: 운명을 받아들이며 최선을 다함
- **대인 관계**: 적까지 포용, 증오 없는 전투
- **에너지 상태**: 고요한 힘, 불꽃 없이도 뜨거움
- **자기 인식**: "나는 수호자다. 죽음 후에도"

#### 신화적 대응

> **라그나로크에서의 토르**
>
> 토르는 세계뱀 요르문간드와 싸워 이깁니다.
> 하지만 독에 맞아 아홉 걸음 후 쓰러집니다.
> 그는 자신의 죽음을 알면서도 싸웠습니다.
> 승리가 아닌, 수호가 목적이었기에.
>

> 델포이 신탁은 헤라클레스에게 말합니다:
> 에우리스테우스 왕을 12년간 섬기고,
> 그가 시키는 과업을 완수하라.
> 헤라클레스는 속죄를 위해 이를 받아들입니다.
>
> **Lv2의 헤라클레스는 속죄를 위해 시련을 감수하는 상태입니다.**

#### 싱크로 신호 (이런 모습이면 Lv2)

- [ ] 과거의 실수를 만회하려 노력
- [ ] 힘든 일을 "해야 한다"며 감수
- [ ] 목표(속죄)가 있어 버틸 수 있음
- [ ] 도움을 받기 시작함
- [ ] 여전히 자기 연민에 빠지기도 함

### Lv3: 통합 (Integration) — "과업을 넘어"

#### 특징

"속죄가 아니라, 이것이 나의 길이다."
```

- **행동 패턴**: 시련을 성장의 기회로 봄
- **대인 관계**: 동료와 함께함 (아르고 원정대)
- **에너지 상태**: 지속적 도전, 회복력 보유
- **자기 인식**: "나는 영웅이다. 시련이 나를 만든다"

#### 신화적 대응

> **12가지 과업 완수**
>
> 헤라클레스는 모든 과업을 완수합니다.
> 네메아의 사자, 레르나이아 히드라, 케르베로스까지.
> 각 과업에서 그는 힘만이 아닌 지혜도 사용합니다.
> 속죄에서 시작했지만, 이제는 자신의 길입니다.
>
> **Lv3의 헤라클레스는 시련을 받아들이고 성장하는 상태입니다.**

#### 싱크로 신호 (이런 모습이면 Lv3)

- [ ] 어려운 일을 "기회"로 봄
- [ ] 과거의 실수를 용서함 (자기 용서)
- [ ] 동료와 함께 더 큰 일을 해냄
- [ ] "해야 한다"에서 "하고 싶다"로
- [ ] 시련 후에 더 강해짐

- **대인 관계**: 모든 고통받는 이의 영웅
- **에너지 상태**: 불멸의 에너지
- **자기 인식**: "나는 길을 보여주는 자"

#### 신화적 대응

## 신화 인물 6: 쿠훌린 (Cú Chulainn)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | Cú Chulainn (쿠훌린) |
| **신화 체계** | 켈트 (아일랜드) |
| **역할** | 얼스터의 영웅, "사냥개" |
| **상징** | 게 볼그(창), 리아스트라드(전투 광란) |

### 동기 프로파일

```
Achievement:  0.95  ████████████████████ (핵심)
Freedom:      0.70  ██████████████
Adventure:    0.60  ████████████
Security:     0.10  ██
Recognition:  0.55  ███████████
Connection:   0.40  ████████
```

### 핵심 특징 3가지

1. **전투 광란 (Ríastrad)** — 전투 시 괴물로 변신
2. **짧지만 영광스러운 삶** — 예언된 요절
3. **명예의 화신** — 명예를 위해 모든 것을 건다

### Lv1: 그림자 (Shadow) — "통제 불능의 괴물"

#### 특징

```
"피가 끓으면 아무것도 보이지 않는다."
```

- **행동 패턴**: 전투 광란 시 적아 구분 못함
- **대인 관계**: 가까운 사람도 해칠 위험
- **에너지 상태**: 폭발적이지만 자기 파괴적
- **자기 인식**: "나는 괴물이다"

#### 신화적 대응

> **첫 전투 광란**
>
> 어린 쿠훌린이 처음 전투 광란에 빠졌을 때,
> 그는 적과 아군을 구분하지 못했습니다.
> 진정시키기 위해 여인들이 맨몸으로 나서고,
> 얼음물 세 통에 담가야 겨우 진정되었습니다.
>
> **Lv1의 쿠훌린은 자신의 힘에 압도당한 상태입니다.**

#### 싱크로 신호 (이런 모습이면 Lv1)

- [ ] 분노 시 "보이지 않는다"
- [ ] 가까운 사람에게 상처를 줌
- [ ] 전투(경쟁) 후 공허함
- [ ] 자기 통제가 어려움
- [ ] "괴물 같은 나"에 대한 두려움

### Lv2: 각성 (Awakening) — "명예를 위한 전사"

#### 특징

```
"명예를 위해 싸운다. 죽음도 두렵지 않다."
```

- **행동 패턴**: 명예로운 전투 추구
- **대인 관계**: 전우에 대한 충성
- **에너지 상태**: 목표(명예) 있을 때 집중
- **자기 인식**: "나는 전사다"

#### 신화적 대응

> **쿨란의 개를 죽인 후**
>
> 어린 세탄타는 대장장이 쿨란의 개를 죽입니다.
> 그는 사과하며 "개가 다 클 때까지 내가 지키겠다"고 합니다.
> 이때부터 그는 "쿨란의 사냥개" 쿠훌린이 됩니다.
> 책임을 지는 것, 이것이 그의 명예입니다.
>
> **Lv2의 쿠훌린은 명예를 중심으로 정체성을 형성하는 상태입니다.**

---

### Lv3: 통합 (Integration) — "홀로 서는 영웅"

#### 특징

```
"모두가 쓰러져도 나는 싸운다."
```

- **행동 패턴**: 압도적 불리함에도 싸움
- **대인 관계**: 동료를 지키기 위해 홀로 싸움
- **에너지 상태**: 극한까지 밀어붙이는 지속력
- **자기 인식**: "얼스터는 내가 지킨다"

#### 신화적 대응

> **쿨리의 소 탈취전**
>
> 얼스터의 전사들이 저주로 쓰러졌을 때,
> 쿠훌린 혼자서 코나트 군대를 막습니다.
> 여울목에서 1:1 결투 규칙을 이용해
> 매일 한 명씩 상대하며 시간을 벌었습니다.
>
> **Lv3의 쿠훌린은 불가능해도 포기하지 않는 상태입니다.**

### Lv4: 초월 (Transcendence) — "서서 죽는 영웅"

#### 특징

```
"죽어서도 나는 전사다."
```

- **행동 패턴**: 죽음을 받아들이며 끝까지 싸움
- **대인 관계**: 적조차 존경함
- **에너지 상태**: 육체를 넘어선 의지
- **자기 인식**: "나는 영원히 얼스터의 영웅"

#### 신화적 대응

> **쿠훌린의 죽음**
>
> 쿠훌린은 저주와 계략에 의해 힘을 잃습니다.
> 창에 맞아 내장이 흘러나오지만,
> 그는 자신을 바위에 묶어 서서 죽습니다.
> 적들은 3일 동안 그에게 다가가지 못했습니다.
> 까마귀가 어깨에 앉아야 죽음을 확인했습니다.
>
> **Lv4의 쿠훌린은 죽음조차 자신의 방식으로 맞이한 상태입니다.**

#### 도달 경지

| 경지 | 발현 |
|------|------|
| 죽음 초월 | 죽어서도 적을 두렵게 함 |
| 선택된 운명 | 짧지만 영광스러운 삶을 선택 |
| 불멸의 명예 | 육체는 죽어도 이름은 영원 |
| 서서 죽음 | 마지막까지 전사로서의 품격 |

## 원형 내 인물 비교표

### 핵심 차이

| 인물 | 힘의 원천 | 싸우는 이유 | 약점 | 초월 방식 |
|------|-----------|-------------|------|-----------|
| **아레스** | 순수한 전투욕 | 싸움 자체 | 목적 없음 | 사랑 발견 |
| **토르** | 보호 본능 | 지키기 위해 | 단순함 | 운명 수용 |
| **항우** | 자존심 | 천하 정복 | 교만 | 패배 속 존엄 |
| **인드라** | 지위 욕구 | 왕좌 유지 | 불안 | 지위 초월 |
| **헤라클레스** | 속죄 욕구 | 죄 씻기 | 광기 | 인간→신 |
| **쿠훌린** | 명예 욕구 | 명예를 위해 | 광란 | 서서 죽음 |

### 레벨별 핵심 과제

| 레벨 | 아레스 | 토르 | 항우 | 인드라 | 헤라클레스 | 쿠훌린 |
|------|--------|------|------|--------|------------|--------|
| **Lv1→2** | 목적 찾기 | 지킬 것 찾기 | 협력 인정 | 증명에서 탈피 | 속죄 시작 | 광란 통제 |
| **Lv2→3** | 패배에서 배움 | 힘의 한계 인정 | 조언 수용 | 책임 수용 | 자기 용서 | 책임 수용 |
| **Lv3→4** | 싸우지 않기 | 운명 수용 | 존엄한 패배 | 지위 초월 | 고통 초월 | 죽음 선택 |

### 당신에게 맞는 인물은?

```
Achievement가 높고...

+ Recognition이 높다 → 인드라, 항우
+ Freedom이 높다 → 아레스, 쿠훌린
+ Connection이 높다 → 토르, 헤라클레스
+ Security가 낮다 → 쿠훌린, 항우
+ Adventure가 높다 → 헤라클레스, 쿠훌린
```

syncRate_ares_lv2 = (

```

## 부록: 레벨별 핵심 질문

### 자가 진단 질문

**Lv1 확인**
- "왜 이기려 하는가?"에 대답할 수 없다
- 지는 것을 견딜 수 없다
- 협력보다 경쟁이 기본값이다

**Lv2 확인**
- 목표가 있으면 집중한다
- 왜 이기려 하는지 알지만, 지면 화가 난다
- 협력은 가능하지만 불편하다

**Lv3 확인**
- 패배에서 배움을 찾을 수 있다
- 경쟁자를 존경할 수 있다
- 팀의 승리가 진심으로 기쁘다

**Lv4 확인**
- 이기지 않아도 괜찮다
- 패배해도 자존감이 흔들리지 않는다
- 싸움보다 평화가 좋다
