## 원형 개요

### 핵심 정체성

**The Sage (현자)**는 **깊이 있는 이해**에서 존재 의미를 찾는 원형입니다.

```
"왜?"라는 질문이 멈추면 영혼이 시든다.
```

### 동기 가중치

| 동기 | 가중치 | 역할 |
|------|--------|------|
| **Mastery** | 0.80 | 핵심 동기 |
| **Creation** | 0.10 | 보조 동기 |
| **Security** | 0.10 | 보조 동기 |

### 원형 일치 조건

```typescript
conditions: {
  mastery >= 70,
  mastery > achievement,
  mastery - adventure >= 15
}
```

### 4단계 성숙도 구조

| 레벨 | 이름 | 핵심 | 신화적 단계 |
|------|------|------|-------------|
| **Lv1** | 그림자 (Shadow) | 아는 척, 지식 과시, 행동 회피 | 은둔/고립 |
| **Lv2** | 각성 (Awakening) | 앎의 즐거움, 탐구 시작 | 수학/여정 |
| **Lv3** | 통합 (Integration) | 앎과 행함의 일치 | 지혜의 발현 |
| **Lv4** | 초월 (Transcendence) | 무지의 지혜, 가르침 | 완성/승천 |

## 신화 인물 1: 제갈량 (諸葛亮)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | 諸葛亮 (제갈량, Zhuge Liang) |
| **신화 체계** | 중국 역사/신화 |
| **역할** | 촉한의 승상, "와룡" |
| **상징** | 깃털 부채, 사륜거, 팔진도 |

### 동기 프로파일

```
Mastery:     0.90  ██████████████████ (핵심)
Achievement: 0.70  ██████████████
Security:    0.50  ██████████
Freedom:     0.30  ██████
Connection:  0.45  █████████
Recognition: 0.40  ████████
```

### 핵심 특징 3가지

1. **천하삼분지계** — 큰 그림을 보는 전략가
2. **충성과 헌신** — 유비에게 바친 일생
3. **모든 것을 계획** — 죽음까지 설계한 지략

---

### Lv1: 그림자 (Shadow) — "융중에서 나오지 않는 자"

#### 특징

```
"아직 때가 아니다. 준비가 더 필요하다."
```

- **행동 패턴**: 끝없는 준비, 실행 회피
- **대인 관계**: 남을 깔봄, "너희는 모른다"
- **에너지 상태**: 생각에 에너지 쏟고 행동에는 무기력
- **자기 인식**: "나는 알고 있다 (하지만 하지 않는다)"

#### 신화적 대응

> **융중에 은둔한 제갈량**
>
> 제갈량은 융중에서 "와룡(臥龍)"이라 불렸습니다.
> 누운 용 — 아직 날지 않은 용.
> 그는 천하를 분석하고 계획했지만,
> 아무도 찾아오지 않으면 영원히 누워있을 뻔했습니다.
>
> **Lv1의 제갈량은 "때를 기다린다"며 행동을 미루는 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 완벽주의 | "더 알아야 해"가 끝나지 않음 |
| 행동 회피 | 분석만 하고 실행은 안 함 |
| 지적 오만 | 남을 무시, "넌 몰라" |
| 고립 | 혼자만의 세계에 갇힘 |

#### 싱크로 신호 (이런 모습이면 Lv1)

- [ ] "준비가 더 필요해"가 3개월 이상
- [ ] 분석은 했지만 결론을 못 내림
- [ ] 남이 틀리면 속으로 깔봄
- [ ] 아는 것을 과시하고 싶은 욕구
- [ ] 실패할까봐 시작을 못함

---

### Lv2: 각성 (Awakening) — "삼고초려에 응한 자"

#### 특징

```
"알고 있다. 이제 써야 할 때다."
```

- **행동 패턴**: 지식을 현실에 적용 시작
- **대인 관계**: 뜻을 함께할 사람 찾음
- **에너지 상태**: 목표 있을 때 활성화
- **자기 인식**: "나는 할 수 있다"

#### 신화적 대응

> **삼고초려 (三顧草廬)**
>
> 유비가 세 번 찾아왔습니다.
> 제갈량은 마침내 융중을 나섭니다.
> "천하삼분지계"를 제시하며 말합니다:
> "장군이 진정 이를 원하신다면, 량이 돕겠습니다."
>
> 지식이 현실로 나온 순간입니다.
>
> **Lv2의 제갈량은 지식을 세상에 펼치기 시작한 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 이론 의존 | 현장과 계획의 괴리 |
| 실행 위임 | 설계만 하고 직접 안 함 |
| 인정 욕구 | 계획이 맞았다는 확인 필요 |
| 책임 분산 | 실패하면 "실행이 잘못됐다" |

#### 싱크로 신호 (이런 모습이면 Lv2)

- [ ] 계획을 세우고 실행은 시작함
- [ ] 결과에 따라 기분이 오르락내리락
- [ ] "내 분석이 맞았다"에 만족감
- [ ] 팀과 함께하지만 여전히 거리감
- [ ] 예상치 못한 상황에 당황

---

### Lv3: 통합 (Integration) — "출사표를 쓴 자"

#### 특징

```
"알고, 행하고, 책임진다."
```

- **행동 패턴**: 설계 + 실행 + 책임의 일치
- **대인 관계**: 팀을 이끌며 함께함
- **에너지 상태**: 지속적 헌신, 회복력 보유
- **자기 인식**: "나는 이 일을 해야 한다"

#### 신화적 대응

> **출사표 (出師表)**
>
> 유비 사후, 제갈량은 북벌을 결심합니다.
> 출사표에서 그는 말합니다:
> "신은 본래 포의(布衣)로서 남양에서 밭을 갈았습니다...
> 선제께서 신을 비천하게 여기지 않으시고..."
>
> 그는 이제 모든 것을 직접 짊어집니다.
> 설계만 하던 자가 전장에 나섭니다.
>
> **Lv3의 제갈량은 앎과 행함이 하나가 된 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 과잉 책임 | 모든 것을 혼자 짊어짐 |
| 완벽주의 | 사소한 것까지 직접 검토 |
| 지침 | 쉬지 못함, 건강 무시 |
| 후계자 부재 | 위임하지 못함 |

#### 싱크로 신호 (이런 모습이면 Lv3)

- [ ] 말한 대로 행동함
- [ ] 실패해도 "내 책임"이라 말함
- [ ] 팀원들이 진심으로 따름
- [ ] 결과와 무관하게 최선을 다함
- [ ] 하지만 쉬지 못함

---

### Lv4: 초월 (Transcendence) — "오장원의 별"

#### 특징

```
"죽고 사는 것은 하늘에 달렸거늘..."
```

- **행동 패턴**: 결과를 초월한 헌신
- **대인 관계**: 적까지 존경함
- **에너지 상태**: 고요한 빛, 육체를 넘어선 의지
- **자기 인식**: "내가 아니어도 괜찮다"

#### 신화적 대응

> **오장원 (五丈原)에서의 최후**
>
> 제갈량은 다섯 번째 북벌에서 병이 듭니다.
> 그는 자신의 죽음을 알았습니다.
> 마지막까지 군무를 처리하고,
> 후계자에게 일을 맡기고 눈을 감습니다.
>
> "죽고 사는 것은 하늘에 달렸거늘,
> 어찌 기도로 바꿀 수 있겠습니까?"
>
> 그의 별이 떨어졌을 때,
> 적장 사마의조차 탄식했습니다:
> "천하의 기재(奇才)로다!"
>
> **Lv4의 제갈량은 결과를 초월하고도 최선을 다한 상태입니다.**

#### 도달 경지

| 경지 | 발현 |
|------|------|
| 결과 초월 | 성공/실패와 무관하게 최선 |
| 운명 수용 | 죽음도 담담히 받아들임 |
| 위임의 지혜 | "내가 아니어도 괜찮다" |
| 적의 존경 | 사마의조차 존경함 |

#### 싱크로 신호 (이런 모습이면 Lv4)

- [ ] 실패해도 평안함
- [ ] 후계자를 기꺼이 키움
- [ ] 내가 없어도 되는 시스템을 만듦
- [ ] 적에게도 배울 것이 있다고 인정
- [ ] "해야 해서"가 아닌 "하고 싶어서"

---

### 레벨 진화 조건

```
Lv1 → Lv2: 융중을 나와 세상에 지식을 펼칠 때
Lv2 → Lv3: 설계뿐 아니라 실행과 책임을 질 때
Lv3 → Lv4: 결과와 무관하게 최선을 다하고, 내가 없어도 되는 것을 받아들일 때
```

### 레벨 퇴화 경고

```
Lv4 → Lv3: 다시 "내가 해야 해"에 집착할 때
Lv3 → Lv2: 실패의 책임을 회피할 때
Lv2 → Lv1: 다시 "아직 준비가 안 됐어"로 돌아갈 때
```

## 신화 인물 2: 아테나 (Athena)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | Athena (아테나) |
| **신화 체계** | 그리스 |
| **역할** | 지혜와 전쟁의 여신 |
| **상징** | 올리브, 올빼미, 창과 방패, 메두사의 머리 |

### 동기 프로파일

```
Mastery:     0.85  █████████████████ (핵심)
Achievement: 0.80  ████████████████
Recognition: 0.50  ██████████
Freedom:     0.40  ████████
Security:    0.45  █████████
Creation:    0.55  ███████████
```

### 핵심 특징 3가지

1. **전략적 지혜** — 생각 후 행동, 승리를 설계
2. **문명의 수호** — 도시와 기술의 수호신
3. **냉철한 판단** — 감정보다 이성

---

### Lv1: 그림자 (Shadow) — "차가운 완벽주의자"

#### 특징

```
"감정은 약점이다. 오직 이성만이 진실이다."
```

- **행동 패턴**: 모든 것을 분석, 감정 억압
- **대인 관계**: 냉정함, 친밀함 어려움
- **에너지 상태**: 통제된 상태, 유연성 부족
- **자기 인식**: "나는 틀리지 않는다"

#### 신화적 대응

> **아라크네를 벌한 아테나**
>
> 아라크네는 직조에서 아테나와 경쟁했습니다.
> 그녀의 작품은 신들의 추문을 담았고,
> 솜씨는 아테나와 대등했습니다.
> 아테나는 분노하여 아라크네를 거미로 바꿉니다.
>
> **Lv1의 아테나는 도전을 용납하지 않는 차가운 완벽주의자입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 감정 억압 | "감정적"이라는 말에 민감 |
| 도전 불용 | 반론을 "틀림"으로 봄 |
| 고립 | 친밀한 관계 형성 어려움 |
| 경직 | 예외를 인정하지 않음 |

#### 싱크로 신호 (이런 모습이면 Lv1)

- [ ] "감정적이지 마"가 기본 태도
- [ ] 틀렸다는 것을 인정하기 어려움
- [ ] 가까운 관계에서도 거리감
- [ ] 반론을 "공격"으로 느낌
- [ ] 완벽하지 않으면 의미 없다고 생각

---

### Lv2: 각성 (Awakening) — "영웅을 돕는 지혜"

#### 특징

```
"나의 지혜로 세상을 바꿀 수 있다."
```

- **행동 패턴**: 지식을 활용해 타인을 도움
- **대인 관계**: 조언자 역할, 거리를 두고 지원
- **에너지 상태**: 목표(성공) 있을 때 활성화
- **자기 인식**: "나는 현명하다"

#### 신화적 대응

> **오디세우스를 돕는 아테나**
>
> 아테나는 오디세우스의 수호신입니다.
> 그녀는 직접 싸우지 않고 조언합니다.
> 변장, 전략, 때를 기다리는 법을 가르칩니다.
> 오디세우스의 승리는 그녀의 지혜의 승리입니다.
>
> **Lv2의 아테나는 지혜로 영웅을 돕는 조언자입니다.**

#### 싱크로 신호 (이런 모습이면 Lv2)

- [ ] 남을 가르치고 조언하는 것을 좋아함
- [ ] 조언이 먹히면 만족, 안 먹히면 답답
- [ ] 거리를 두고 지원함
- [ ] 직접 나서기보다 뒤에서 설계
- [ ] "내 덕분"이라는 생각이 있음

---

### Lv3: 통합 (Integration) — "지혜와 용기의 일치"

#### 특징

```
"알고, 직접 나서고, 함께한다."
```

- **행동 패턴**: 조언만이 아닌 직접 행동
- **대인 관계**: 동료로서 함께함
- **에너지 상태**: 지속 가능한 헌신
- **자기 인식**: "지혜는 행동으로 완성된다"

#### 신화적 대응

> **기간토마키아의 아테나**
>
> 거인들과의 전쟁에서 아테나는 직접 싸웁니다.
> 엔켈라도스를 쓰러뜨리고 시칠리아를 그 위에 던집니다.
> 조언자에서 전사로, 지혜가 행동이 됩니다.
>
> **Lv3의 아테나는 지혜와 용기가 하나 된 상태입니다.**

#### 싱크로 신호 (이런 모습이면 Lv3)

- [ ] 필요하면 직접 나섬
- [ ] 동료로서 함께 싸움
- [ ] 실패해도 "다음엔 다르게"
- [ ] 감정도 데이터로 인정함
- [ ] 완벽하지 않아도 진행함

---

### Lv4: 초월 (Transcendence) — "문명의 어머니"

#### 특징

```
"나의 지혜는 모두의 것이다."
```

- **행동 패턴**: 지혜를 나눔, 문명을 건설
- **대인 관계**: 모든 이의 스승
- **에너지 상태**: 영원한 영향력
- **자기 인식**: "나는 길을 닦는 자"

#### 신화적 대응

> **아테네에 올리브를 준 아테나**
>
> 포세이돈과 아테네의 수호신 경쟁에서
> 포세이돈은 샘물을, 아테나는 올리브 나무를 줍니다.
> 올리브는 기름, 음식, 목재, 평화를 상징합니다.
> 아테나는 승리했고, 도시는 그녀의 이름을 땄습니다.
>
> **Lv4의 아테나는 자신의 지혜를 문명으로 남긴 상태입니다.**

#### 도달 경지

| 경지 | 발현 |
|------|------|
| 지혜 공유 | 지식을 독점하지 않음 |
| 문명 건설 | 개인이 아닌 시스템을 만듦 |
| 영원한 영향 | 이름이 도시가 됨 |
| 냉철함과 따뜻함 | 이성과 감정의 균형 |

---

### 레벨 진화 조건

```

- **행동 패턴**: 지식으로 질서 유지에 참여
- **대인 관계**: 조정자, 중재자
- **에너지 상태**: 지속적 책임감
- **자기 인식**: "나는 마아트를 지킨다"

#### 신화적 대응

> **호루스와 세트의 분쟁을 중재한 토트**
>
> 토트는 호루스와 세트의 싸움에서 중재합니다.
> 기록자에서 참여자로.
> 그의 지혜가 분쟁 해결에 쓰입니다.
>
> **Lv3의 토트는 지혜로 질서를 지키는 데 참여하는 상태입니다.**

#### 싱크로 신호 (이런 모습이면 Lv3)

- [ ] 갈등 상황에서 중재함
- [ ] 지식을 정의/질서에 활용
- [ ] 방관자에서 참여자로
- [ ] 책임을 받아들임
- [ ] "옳은 것"을 위해 나섬

### Lv4: 초월 (Transcendence) — "마아트의 수호자"

```

- **행동 패턴**: 우주적 질서 유지에 헌신

> **Lv4의 토트는 우주적 진실을 기록하고 지키는 상태입니다.**

### 핵심 특징 3가지

2. **순수한 추구** — 물질보다 지혜, 명예보다 진실
3. **흐르는 지혜** — 강처럼 자연스럽게 흐름

### Lv1: 그림자 (Shadow) — "상아탑의 예술가"

#### 특징

```
"세상은 속되다. 나의 예술만이 순수하다."
```

- **행동 패턴**: 현실과 단절, 예술에 도피
- **대인 관계**: 엘리트주의, "너희는 몰라"
- **에너지 상태**: 영감 있을 때만 활성화
- **자기 인식**: "나만이 진정한 예술가"

#### 신화적 대응

> **브라흐마를 피하는 사라스와티**
>
> 사라스와티는 브라흐마의 창조물이자 배우자입니다.
> 하지만 그녀는 종종 그를 피합니다.
> 순수를 지키기 위해? 아니면 관계를 회피하기 위해?
>
> **Lv1의 사라스와티는 순수를 핑계로 세상을 피하는 상태입니다.**

#### 싱크로 신호 (이런 모습이면 Lv1)

- [ ] "세상이 나를 이해 못해"
- [ ] 예술/학문에 도피
- [ ] 현실적 책임 회피
- [ ] 엘리트주의, 대중 멸시
- [ ] 영감 없으면 무기력

### Lv2: 각성 (Awakening) — "가르치기 시작한 자"

#### 특징

```
"나의 지혜는 나누어야 한다."
```

- **행동 패턴**: 가르치고 공유 시작
- **대인 관계**: 스승 역할 시작
- **에너지 상태**: 목표(전달) 있을 때 활성화
- **자기 인식**: "나의 예술은 가르침이다"

#### 신화적 대응

> **인간에게 지식을 주는 사라스와티**
>
> 사라스와티는 인간에게 언어, 음악, 예술을 줍니다.
> 그녀의 축복 없이는 학문도 예술도 없습니다.
> 상아탑에서 내려와 가르치기 시작합니다.
>
> **Lv2의 사라스와티는 지식을 나누기 시작한 상태입니다.**

### Lv3: 통합 (Integration) — "흐르는 강"

#### 특징

```
"지혜는 막히면 썩는다. 흘러야 한다."
```

- **행동 패턴**: 자연스럽게 지식이 흐름
- **대인 관계**: 모든 이에게 열림
- **에너지 상태**: 지속적 창조와 공유
- **자기 인식**: "나는 강이다"

#### 신화적 대응

> **사라스와티 강**
>
> 그녀의 이름은 "흐르는 것"을 의미합니다.
> 지혜는 강처럼 흘러야 합니다.
> 막히면 고이고, 고이면 썩습니다.
>
> **Lv3의 사라스와티는 자연스럽게 지혜를 흘려보내는 상태입니다.**

### Lv4: 초월 (Transcendence) — "순수한 빛"

#### 특징

```
"나는 지혜 그 자체다."
```

- **행동 패턴**: 존재 자체가 가르침
- **대인 관계**: 모든 학습자의 원천
- **에너지 상태**: 영원한 영감
- **자기 인식**: "나와 지혜는 하나"

#### 신화적 대응

> **백조 위의 사라스와티**
>
> 사라스와티는 백조 위에 앉아 있습니다.
> 백조는 물과 우유를 분리할 수 있다고 합니다.
> 진실과 거짓, 지혜와 무지를 구분하는 능력.
> 그녀는 그 분별 자체입니다.
>
> **Lv4의 사라스와티는 지혜 그 자체가 된 상태입니다.**
