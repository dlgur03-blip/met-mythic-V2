## 원형 개요

### 핵심 정체성

**The Creator (창조자)**는 **새로운 것을 만드는 것**에서 존재 의미를 찾는 원형입니다.

```
"없던 것을 있게 하는 순간, 나는 살아있다."
```

### 동기 가중치

| 동기 | 가중치 | 역할 |
|------|--------|------|
| **Creation** | 0.70 | 핵심 동기 |
| **Mastery** | 0.15 | 보조 동기 |
| **Freedom** | 0.15 | 보조 동기 |

### 원형 일치 조건

```typescript
conditions: {
  creation >= 70,
  creation > security,
  freedom >= 50
}
```

### 4단계 성숙도 구조

| 레벨 | 이름 | 핵심 | 신화적 단계 |
|------|------|------|-------------|
| **Lv1** | 그림자 (Shadow) | 미완의 집착, 파괴적 창조 | 혼돈/미완성 |
| **Lv2** | 각성 (Awakening) | 창조의 즐거움, 첫 완성 | 첫 작품 |
| **Lv3** | 통합 (Integration) | 책임지는 창조, 지속성 | 걸작/대작 |
| **Lv4** | 초월 (Transcendence) | 창조를 통한 연결, 유산 | 신화적 창조 |

## 신화 인물 1: 헤파이스토스 (Hephaestus)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | Hephaestus (헤파이스토스) |
| **신화 체계** | 그리스 |
| **역할** | 대장장이의 신, 기술과 공예의 신 |
| **상징** | 망치, 모루, 집게, 화산 |

### 동기 프로파일

```
Creation:    0.95  ███████████████████ (핵심)
Mastery:     0.80  ████████████████
Security:    0.50  ██████████
Recognition: 0.30  ██████
Freedom:     0.40  ████████
Connection:  0.35  ███████
```

### 핵심 특징 3가지

1. **불완전함 속의 완벽** — 절름발이지만 최고의 장인
2. **작품으로 말한다** — 말보다 만든 것으로 증명
3. **고독한 창조** — 혼자서 화산 속에서 작업

---

### Lv1: 그림자 (Shadow) — "버림받은 대장장이"

#### 특징

```
"아무도 나를 원하지 않았다. 내 작품만이 나를 증명한다."
```

- **행동 패턴**: 인정받기 위한 강박적 창조
- **대인 관계**: 고립, 거부에 대한 두려움
- **에너지 상태**: 작업 중독, 번아웃 반복
- **자기 인식**: "나는 불완전하다. 작품만이 완전해야 한다"

#### 신화적 대응

> **올림포스에서 버림받은 헤파이스토스**
>
> 헤라는 못생긴 아들을 올림포스에서 던졌습니다.
> 바다로 떨어진 그는 테티스에게 구해집니다.
> 9년간 바다 밑 동굴에서 홀로 일합니다.
> 버림받은 상처가 창조의 연료가 됩니다.
>
> **Lv1의 헤파이스토스는 상처를 작품으로 메우려는 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 인정 강박 | 작품으로 사랑받으려 함 |
| 완벽주의 | 완벽하지 않으면 가치 없음 |
| 고립 | 사람보다 작업실 |
| 작업 중독 | 쉬면 불안, 멈추지 못함 |

#### 싱크로 신호 (이런 모습이면 Lv1)

- [ ] 작품이 인정받지 못하면 자존감 추락
- [ ] 완성하지 못한 프로젝트가 쌓임
- [ ] 혼자 작업하는 것이 편함
- [ ] "이것만 완성하면"이 끝나지 않음
- [ ] 사람보다 작업이 우선

---

### Lv2: 각성 (Awakening) — "경이로운 작품들"

#### 특징

```
"내가 만든 것이 세상을 놀라게 한다."
```

- **행동 패턴**: 창조의 즐거움 발견, 완성의 기쁨
- **대인 관계**: 작품을 통해 연결 시작
- **에너지 상태**: 영감 있을 때 폭발적
- **자기 인식**: "나는 만들 수 있다"

#### 신화적 대응

> **신들의 무기를 만든 헤파이스토스**
>
> 제우스의 번개, 포세이돈의 삼지창, 하데스의 투구.
> 아킬레우스의 방패, 헤르메스의 날개 신발.
> 헤파이스토스의 작품은 신들도 탐냅니다.
>
> **Lv2의 헤파이스토스는 창조의 힘을 깨달은 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 영감 의존 | 영감 없으면 무기력 |
| 외부 평가 | 반응에 따라 기분 변화 |
| 다음 작품 | 완성 후 바로 다음으로 |
| 협력 어려움 | 혼자가 편함 |

#### 싱크로 신호 (이런 모습이면 Lv2)

- [ ] 완성의 기쁨을 알게 됨
- [ ] 작품이 인정받으면 기쁨
- [ ] 영감이 오면 밤새 작업
- [ ] 협력은 가능하지만 불편
- [ ] "다음엔 더 잘 만들겠다"

---

### Lv3: 통합 (Integration) — "화산의 대장간"

#### 특징

```
"만드는 것은 책임이다. 내 작품은 세상에 영향을 준다."
```

- **행동 패턴**: 지속적 창조, 책임감
- **대인 관계**: 협력 가능, 도제 양성
- **에너지 상태**: 지속 가능한 창작 리듬
- **자기 인식**: "나는 장인이다"

#### 신화적 대응

> **아프로디테와의 결혼**
>
> 헤파이스토스는 가장 아름다운 여신과 결혼합니다.
> 불완전한 그가 완벽한 미와 결합합니다.
> 관계는 복잡했지만, 그는 성장합니다.
> 고립에서 나와 연결을 배웁니다.
>
> **Lv3의 헤파이스토스는 창조와 관계를 함께 다루는 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 과잉 생산 | 쉬지 못함 |
| 결과 집착 | 과정보다 결과물 |
| 완벽주의 지속 | "더 좋게 더 좋게" |

#### 싱크로 신호 (이런 모습이면 Lv3)

- [ ] 작품의 사회적 영향을 고려함
- [ ] 후배/도제를 가르침
- [ ] 협력이 자연스러워짐
- [ ] 완성 후 쉴 수 있음
- [ ] "충분히 좋다"를 받아들임

---

### Lv4: 초월 (Transcendence) — "자동인형의 창조자"

#### 특징

```
"나의 창조물은 스스로 생명을 가진다."
```

- **행동 패턴**: 창조를 통한 새 생명 부여
- **대인 관계**: 작품이 곧 연결
- **에너지 상태**: 고요한 장인 정신
- **자기 인식**: "나는 생명을 주는 자"

#### 신화적 대응

> **탈로스와 황금 시녀들**
>
> 헤파이스토스는 청동 거인 탈로스를 만들어
> 크레타를 지키게 했습니다.
> 황금 시녀들은 스스로 움직이며 그를 도왔습니다.
> 그의 창조물은 단순한 물건이 아닌 생명입니다.
>
> **Lv4의 헤파이스토스는 창조를 통해 생명을 주는 상태입니다.**

#### 도달 경지

| 경지 | 발현 |
|------|------|
| 생명 부여 | 작품이 스스로 움직임 |
| 불완전함 수용 | 자신의 절름발이를 받아들임 |
| 창조의 평화 | 인정 없이도 만듦 |
| 유산 | 작품이 영원히 남음 |

#### 싱크로 신호 (이런 모습이면 Lv4)

- [ ] 인정받지 못해도 만듦
- [ ] 작품이 나를 떠나도 괜찮음
- [ ] 불완전한 나를 받아들임
- [ ] 창조가 곧 연결임을 앎
- [ ] 작품에 생명을 불어넣음

---

### 레벨 진화 조건

```
Lv1 → Lv2: 인정이 아닌 창조 자체의 기쁨을 발견할 때
Lv2 → Lv3: 창조의 책임과 지속성을 받아들일 때
Lv3 → Lv4: 창조를 통해 연결하고, 작품에 생명을 줄 때
```

### 레벨 퇴화 경고

```
Lv4 → Lv3: 다시 외부 인정에 집착할 때
Lv3 → Lv2: 협력을 거부하고 고립될 때
Lv2 → Lv1: 완성하지 못하고 상처에 빠질 때
```

## 신화 인물 2: 프로메테우스 (Prometheus)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | Prometheus (프로메테우스) |
| **신화 체계** | 그리스 |
| **역할** | 인류의 창조자, 불을 훔친 타이탄 |
| **상징** | 불, 횃불, 독수리, 간 |

### 동기 프로파일

```
Creation:    0.90  ██████████████████ (핵심)
Freedom:     0.80  ████████████████
Connection:  0.60  ████████████
Security:    0.10  ██
Mastery:     0.65  █████████████
Recognition: 0.40  ████████
```

### 핵심 특징 3가지

1. **반역적 창조** — 규칙을 어기고 새로움을 만듦
2. **희생적 사랑** — 인류를 위해 영원한 고통
3. **선견지명** — 이름 자체가 "먼저 생각하는 자"

---

### Lv1: 그림자 (Shadow) — "통제할 수 없는 창조"

#### 특징

```
"만들어야 한다. 결과가 어떻든."
```

- **행동 패턴**: 결과 고려 없는 창조
- **대인 관계**: 자신의 비전에 타인을 끌어들임
- **에너지 상태**: 충동적, 불안정
- **자기 인식**: "나만이 할 수 있다"

#### 신화적 대응

> **진흙으로 인간을 만든 프로메테우스**
>
> 프로메테우스는 진흙으로 인간을 빚었습니다.
> 하지만 그들은 약하고 무력했습니다.
> 동물에게도 못 미치는 존재.
> 창조는 했지만, 책임은 생각지 못했습니다.
>
> **Lv1의 프로메테우스는 결과를 고려하지 않고 창조하는 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 무책임한 창조 | 만들고 방치 |
| 규칙 무시 | "규칙 따위" |
| 결과 경시 | 과정만 중요 |
| 타인 희생 | 비전을 위해 남을 이용 |

#### 싱크로 신호 (이런 모습이면 Lv1)

- [ ] 만들고 나서 관리를 안 함
- [ ] 규칙 어기는 것에 쾌감
- [ ] "결과는 나중에" 태도
- [ ] 타인을 프로젝트에 끌어들이고 방치
- [ ] 책임지지 않는 혁신

---

### Lv2: 각성 (Awakening) — "불을 훔친 자"

#### 특징

```
"그들에게 필요하다. 내가 주겠다."
```

- **행동 패턴**: 목적 있는 창조, 타인을 위함
- **대인 관계**: 보호자 역할 시작
- **에너지 상태**: 사명감으로 활성화
- **자기 인식**: "나는 그들을 도울 수 있다"

#### 신화적 대응

> **신들의 불을 훔친 프로메테우스**
>
> 인간이 추위와 어둠 속에서 고통받는 것을 보고
> 프로메테우스는 올림포스에서 불을 훔칩니다.
> 회향 줄기에 불씨를 숨겨 인간에게 줍니다.
> 이제 인간은 요리하고, 도구를 만들고, 문명을 세웁니다.
>
> **Lv2의 프로메테우스는 타인을 위해 창조하기 시작한 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 영웅 콤플렉스 | "나만이 구할 수 있다" |
| 결과 무시 | 좋은 의도, 나쁜 결과 |
| 독단 | 물어보지 않고 줌 |
| 대가 경시 | 자신이 치를 대가 무시 |

#### 싱크로 신호 (이런 모습이면 Lv2)

- [ ] 남을 위해 만든다는 명분
- [ ] 규칙을 어겨도 좋은 이유가 있음
- [ ] "이건 필요한 거야"
- [ ] 결과보다 동기에 집중
- [ ] 대가를 생각하지 않음

---

### Lv3: 통합 (Integration) — "대가를 치르는 창조자"

#### 특징

```
"알고 있었다. 그래도 해야 했다. 그리고 책임진다."
```

- **행동 패턴**: 결과를 알면서도 창조, 책임 수용
- **대인 관계**: 진정한 희생, 조건 없는 사랑
- **에너지 상태**: 고통 속에서도 지속
- **자기 인식**: "나는 선택했고, 감당한다"

#### 신화적 대응

> **코카서스에 묶인 프로메테우스**
>
> 제우스는 프로메테우스를 바위에 묶습니다.
> 매일 독수리가 그의 간을 파먹고,
> 밤이 되면 다시 자랍니다.
> 영원한 고통. 하지만 그는 후회하지 않습니다.
>
> "다시 해도 같은 선택을 할 것이다."
>
> **Lv3의 프로메테우스는 대가를 알면서도 선택하고 감당하는 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 순교자 콤플렉스 | 고통에 의미 부여 과잉 |
| 도움 거부 | "나 혼자 감당해야 해" |
| 피해자 정체성 | 희생을 자아로 삼음 |

#### 싱크로 신호 (이런 모습이면 Lv3)

- [ ] 결과를 알면서도 선택함
- [ ] 대가를 치르고도 후회 없음
- [ ] "내 선택이다"라고 말할 수 있음
- [ ] 고통 속에서도 의미를 찾음
- [ ] 도움을 받아들일 수 있음

---

### Lv4: 초월 (Transcendence) — "인류의 아버지"

#### 특징

```
"나의 고통이 그들의 빛이 되었다."
```

- **행동 패턴**: 고통을 초월한 사랑
- **대인 관계**: 모든 인류의 은인
- **에너지 상태**: 영원한 불꽃
- **자기 인식**: "나는 그들 안에 살아있다"

#### 신화적 대응

> **헤라클레스에게 구출된 프로메테우스**
>
> 수천 년 후, 헤라클레스가 프로메테우스를 구합니다.
> 독수리를 쏘고 사슬을 끊습니다.
> 인간이 만든 영웅이 창조자를 구합니다.
> 프로메테우스가 준 불로 자란 인류가
> 이제 그를 구합니다.
>
> **Lv4의 프로메테우스는 자신의 창조물에게 구원받는 상태입니다.**

#### 도달 경지

| 경지 | 발현 |
|------|------|

> **Lv3의 프타는 생각-말-행동이 하나가 된 상태입니다.**

### Lv4: 초월 (Transcendence) — "만물의 심장과 혀"

#### 특징

```
"나의 생각이 곧 우주다."
```

- **행동 패턴**: 존재 자체가 창조
- **대인 관계**: 모든 창조의 원천
- **에너지 상태**: 고요한 전능
- **자기 인식**: "나는 시작이자 끝"

#### 신화적 대응

> **모든 신의 근원인 프타**
>
> 멤피스 신학에서 프타는 가장 오래된 신입니다.
> 다른 모든 신은 프타의 생각에서 나왔습니다.
> 그는 창조하지만 움직이지 않습니다.
> 고요 속에서 우주가 탄생합니다.
>
> **Lv4의 프타는 존재 자체가 창조의 근원인 상태입니다.**

## 신화 인물 6: 이자나기 & 이자나미 (Izanagi & Izanami)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | Izanagi & Izanami (이자나기 & 이자나미) |
| **신화 체계** | 일본 |
| **역할** | 일본 열도와 신들의 창조자 |
| **상징** | 아마노누보코(천상의 창), 섬들 |

### 동기 프로파일

```
Creation:    0.80  ████████████████ (핵심)
Connection:  0.70  ██████████████
Adventure:   0.50  ██████████
Security:    0.50  ██████████
Freedom:     0.40  ████████

```

### 핵심 특징 3가지

1. **함께하는 창조** — 부부가 함께 창조
2. **시행착오** — 첫 아이가 실패, 다시 시도
3. **상실과 재창조** — 이자나미의 죽음 후에도 계속

### Lv1: 그림자 (Shadow) — "실패한 첫 창조"

#### 특징

```
"뭔가 잘못됐다. 누구 탓인가?"
```

- **행동 패턴**: 실패에 대한 비난
- **대인 관계**: 책임 회피, 갈등
- **에너지 상태**: 좌절과 분노
- **자기 인식**: "내 잘못이 아니야"

#### 신화적 대응

> **히루코(수초 아이)의 탄생**
>
> 이자나기와 이자나미가 처음 낳은 아이는
> 히루코 — 뼈 없는 수초 같은 아이였습니다.
> 그들은 아이를 배에 태워 떠나보냅니다.
> 무엇이 잘못되었는지 신들에게 물었습니다.
>
> **Lv1의 이자나기/이자나미는 실패에 당황하고 비난하는 상태입니다.**

#### 싱크로 신호 (이런 모습이면 Lv1)

- [ ] 실패하면 남 탓
- [ ] 첫 시도가 망하면 포기
- [ ] "이건 네 잘못이야"
- [ ] 잘못된 결과물을 방치
- [ ] 함께하지만 갈등

---

### Lv2: 각성 (Awakening) — "다시 시작하는 창조"

#### 특징

```
"잘못된 것을 바로잡고 다시 시작한다."
```

- **행동 패턴**: 실패에서 배움, 재시도
- **대인 관계**: 협력으로 극복
- **에너지 상태**: 회복력
- **자기 인식**: "다시 할 수 있다"

#### 신화적 대응

> **의식을 바로잡은 이자나기와 이자나미**
>
> 신들은 알려줍니다: 여자가 먼저 말해서 그렇다.
> (당시 신화의 관점으로) 의식을 바로잡고 다시 합니다.
> 이번에는 건강한 섬들과 신들이 태어납니다.
>
> **Lv2의 이자나기/이자나미는 실패를 수정하고 재시도하는 상태입니다.**

#### 싱크로 신호 (이런 모습이면 Lv2)

- [ ] 실패에서 배움
- [ ] 다시 시도함
- [ ] 협력하여 극복
- [ ] "다음엔 다르게"
- [ ] 회복력이 있음

---

### Lv3: 통합 (Integration) — "일본을 창조한 부부"

#### 특징

```
"함께 만들고, 함께 책임진다."
```

- **행동 패턴**: 협력적 창조, 역할 분담
- **대인 관계**: 진정한 파트너십
- **에너지 상태**: 지속적 창조
- **자기 인식**: "우리는 함께 창조한다"

#### 신화적 대응

> **팔백만 신을 낳은 이자나기와 이자나미**
>
> 그들은 일본의 섬들을 창조하고,
> 바다, 강, 바람, 나무, 산의 신들을 낳습니다.
> 마지막으로 불의 신 카구츠치를 낳다가
> 이자나미가 화상으로 죽습니다.
>
> **Lv3의 이자나기/이자나미는 함께 창조하며 대가도 함께 치르는 상태입니다.**

### Lv4: 초월 (Transcendence) — "이별 후에도 계속되는 창조"

#### 특징

```
"그녀가 떠나도, 창조는 계속된다."
```

- **행동 패턴**: 상실을 딛고 창조 지속
- **대인 관계**: 이별을 받아들임
- **에너지 상태**: 슬픔 속에서도 지속
- **자기 인식**: "혼자서도 창조할 수 있다"

#### 신화적 대응

> **황천에서 돌아온 이자나기**
>
> 이자나기는 이자나미를 되살리려 황천으로 갑니다.
> 하지만 그녀는 이미 죽은 자의 모습.
> 그는 도망치고, 그녀와 이별합니다.
> "나는 매일 1000명을 죽이겠다"
> "나는 매일 1500명을 태어나게 하겠다"
> 
> 이별 후에도 이자나기는 창조를 계속합니다.
> 코를 씻을 때 스사노오, 눈을 씻을 때 아마테라스가 태어납니다.
>
> **Lv4의 이자나기는 상실을 받아들이고도 창조를 계속하는 상태입니다.**

#### 도달 경지

| 경지 | 발현 |
|------|------|
| 상실 초월 | 이별 후에도 계속 |
| 혼자서도 가능 | 파트너 없이도 창조 |
| 삶과 죽음 | 둘 다 창조의 일부 |
| 영원한 순환 | 죽음보다 탄생이 많음 |
