## 원형 개요

### 핵심 정체성

**The Rebel (반역자)**는 **자유와 변혁**에서 존재 의미를 찾는 원형입니다.

```
"통제받느니 차라리 불태우겠다."
```

### 동기 가중치

| 동기 | 가중치 | 역할 |
|------|--------|------|
| **Freedom** | 0.70 | 핵심 동기 |
| **Creation** | 0.15 | 보조 동기 |
| **Adventure** | 0.15 | 보조 동기 |

### 원형 일치 조건

```typescript
conditions: {
  freedom >= 70,
  freedom > security,
  security <= 40
}
```

### 4단계 성숙도 구조

| 레벨 | 이름 | 핵심 | 신화적 단계 |
|------|------|------|-------------|
| **Lv1** | 그림자 (Shadow) | 무목적 파괴, 반항을 위한 반항 | 혼돈/파괴 |
| **Lv2** | 각성 (Awakening) | 무엇에 반항하는지 인식 | 저항 시작 |
| **Lv3** | 통합 (Integration) | 건설적 반역, 대안 제시 | 혁명가 |
| **Lv4** | 초월 (Transcendence) | 자유 그 자체, 해방자 | 신화적 해방 |

## 신화 인물 1: 손오공 (孫悟空)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | 孫悟空 (손오공, Sun Wukong) |
| **신화 체계** | 중국 (서유기) |
| **역할** | 제천대성, 천상을 뒤흔든 원숭이 |
| **상징** | 여의봉, 근두운, 72변화, 금고아 |

### 동기 프로파일

```
Freedom:     0.95  ███████████████████ (핵심)
Adventure:   0.80  ████████████████
Achievement: 0.70  ██████████████
Security:    0.05  █
Recognition: 0.60  ████████████
Mastery:     0.55  ███████████
```

### 핵심 특징 3가지

1. **천상에 도전** — 옥황상제도 두렵지 않음
2. **불복(不服)** — "나는 내 규칙대로 산다"
3. **속박과 해방** — 금고아(긴고아)와의 싸움

---

### Lv1: 그림자 (Shadow) — "대란천궁(大鬧天宮)"

#### 특징

```
"내 맘대로 할 거야. 누가 막든."
```

- **행동 패턴**: 무분별한 파괴, 규칙 무시
- **대인 관계**: 권위 전부 거부
- **에너지 상태**: 폭발적, 통제 불능
- **자기 인식**: "나는 제천대성, 누구도 막지 못한다"

#### 신화적 대응

> **천상을 뒤집은 손오공**
>
> 손오공은 천상에서 난동을 부렸습니다.
> 반도(蟠桃)를 훔쳐 먹고, 금단(金丹)을 삼키고,
> 10만 천병천장과 싸웠습니다.
> 아무도 막지 못했습니다.
> 하지만 그 끝은 오행산 아래 500년.
>
> **Lv1의 손오공은 무목적한 파괴자 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 무목적 파괴 | 왜 반항하는지 모름 |
| 결과 무시 | 결과를 생각 안 함 |
| 고립 | 모두를 적으로 만듦 |
| 자기 파괴 | 결국 자신이 갇힘 |

#### 싱크로 신호 (이런 모습이면 Lv1)

- [ ] 규칙이면 무조건 반대
- [ ] 왜 반항하는지 설명 못함
- [ ] 결과를 생각 안 함
- [ ] 모든 권위를 적으로 봄
- [ ] 파괴 후 공허함

---

### Lv2: 각성 (Awakening) — "삼장법사를 만난 원숭이"

#### 특징

```
"나는 자유롭고 싶다. 하지만 이 여정에도 의미가 있다."
```

- **행동 패턴**: 목적 있는 반항, 선택적 복종
- **대인 관계**: 스승과 동료 수용
- **에너지 상태**: 방향성 있는 에너지
- **자기 인식**: "나는 제자이기도 하다"

#### 신화적 대응

> **금고아를 쓴 손오공**
>
> 삼장법사는 손오공에게 금고아를 씌웁니다.
> 주문을 외우면 머리가 조여옵니다.
> 손오공은 분노하지만, 결국 받아들입니다.
> 완전한 자유는 없다는 것을 배우기 시작합니다.
>
> **Lv2의 손오공은 속박 속에서도 의미를 찾기 시작한 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 갈등 | 자유와 의무 사이에서 |
| 분노 | 금고아에 대한 원망 |
| 타협 어려움 | 규칙을 여전히 싫어함 |
| 의존 | 삼장 없으면 방향 없음 |

#### 싱크로 신호 (이런 모습이면 Lv2)

- [ ] 어느 정도 규칙을 받아들임
- [ ] 하지만 여전히 불편함
- [ ] 목적이 생김 (경을 가져오는 여정)
- [ ] 동료와 함께함
- [ ] 자유와 의무 사이에서 갈등

### Lv3: 통합 (Integration) — "투전승불(鬪戰勝佛)"

#### 특징

```
"나는 싸운다. 하지만 지키기 위해 싸운다."
```

- **행동 패턴**: 목적 있는 싸움, 보호를 위한 반항
- **대인 관계**: 진정한 충성, 선택한 복종
- **에너지 상태**: 지속 가능한 전투력
- **자기 인식**: "나는 싸우는 자, 지키는 자"

#### 신화적 대응

> **삼장을 지키는 손오공**
>
> 서역 여정에서 손오공은 수많은 요괴와 싸웁니다.
> 더 이상 자신을 위해서가 아닙니다.
> 삼장법사를, 동료들을, 여정을 지키기 위해.
> 반항은 보호가 되었습니다.
>
> **Lv3의 손오공은 싸움의 목적을 찾은 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 과잉 보호 | 모든 것을 혼자 싸움 |
| 정체성 혼란 | 반역자인가 수호자인가 |
| 금고아 원망 | 여전히 속박이 불편 |

#### 싱크로 신호 (이런 모습이면 Lv3)

- [ ] 무엇을 위해 싸우는지 명확
- [ ] 지키고 싶은 것이 있음
- [ ] 규칙도 필요할 때가 있음을 앎
- [ ] 동료를 위해 싸움
- [ ] 선택적으로 복종함

---

### Lv4: 초월 (Transcendence) — "투전승불(鬪戰勝佛)"

#### 특징

```
"금고아가 없어도 나는 나다. 자유는 내 안에 있다."
```

- **행동 패턴**: 내면의 자유, 외적 속박 초월
- **대인 관계**: 스승과 동등, 모든 존재와 연결
- **에너지 상태**: 고요한 힘
- **자기 인식**: "나는 부처가 되었다"

#### 신화적 대응

> **부처가 된 손오공**
>
> 여정이 끝나고, 손오공은 "투전승불"이 됩니다.
> 싸워서 이긴 부처.
> 금고아는 더 이상 필요 없습니다.
> 그는 스스로 자유롭기 때문입니다.
>
> 반역자가 해방자가 되고,
> 해방자가 깨달은 자가 됩니다.
>
> **Lv4의 손오공은 진정한 자유를 얻은 상태입니다.**

#### 도달 경지

| 경지 | 발현 |
|------|------|
| 내면의 자유 | 외적 속박과 무관 |
| 금고아 초월 | 더 이상 필요 없음 |
| 성불 | 반역자에서 부처로 |
| 평화로운 힘 | 싸우되 분노 없음 |

#### 싱크로 신호 (이런 모습이면 Lv4)

- [ ] 규칙에 반항할 필요 없음 (이미 자유)
- [ ] 속박을 초월함
- [ ] 싸우되 평화로움
- [ ] 반항이 아닌 해방을 줌
- [ ] 존재 자체가 자유

---

### 레벨 진화 조건

```
Lv1 → Lv2: 무엇을 위해 반항하는지 알게 될 때
Lv2 → Lv3: 지키고 싶은 것을 위해 싸울 때
Lv3 → Lv4: 외적 속박 없이도 자유로울 때
```

### 레벨 퇴화 경고

```
Lv4 → Lv3: 다시 속박에 반응할 때
Lv3 → Lv2: 목적을 잃고 갈등에 빠질 때
Lv2 → Lv1: 모든 규칙을 다시 거부할 때
```

## 신화 인물 2: 로키 (Loki)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | Loki (로키) |
| **신화 체계** | 북유럽 |
| **역할** | 트릭스터, 혼돈의 신 |
| **상징** | 변신, 불, 뱀, 장난 |

### 동기 프로파일

```
Freedom:     0.90  ██████████████████ (핵심)
Creation:    0.70  ██████████████
Adventure:   0.60  ████████████
Connection:  0.20  ████
Security:    0.15  ███
Recognition: 0.50  ██████████
```

### 핵심 특징 3가지

1. **변신의 달인** — 어떤 형태로든 변할 수 있음
2. **혼돈과 창조** — 파괴하고 새로 만듦
3. **애증의 관계** — 신들을 돕고, 신들을 배신함

---

### Lv1: 그림자 (Shadow) — "배신자"

#### 특징

```
"재미있으면 그만이야. 결과 따위."
```

- **행동 패턴**: 무책임한 장난, 결과 무시
- **대인 관계**: 배신과 이용
- **에너지 상태**: 혼돈, 불안정
- **자기 인식**: "나는 누구 편도 아니다"

#### 신화적 대응

> **발두르를 죽인 로키**
>
> 로키는 발두르를 죽게 만들었습니다.
> 장님 호드에게 겨우살이 화살을 쥐어주고,
> 무적의 발두르를 죽게 했습니다.
> 장난인지, 악의인지, 그 자신도 모릅니다.
>
> **Lv1의 로키는 결과를 고려하지 않는 파괴자입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 무책임 | 결과를 생각 안 함 |
| 배신 | 누구도 믿지 않고 배신 |
| 고립 | 결국 모두에게 버림받음 |
| 자기 파괴 | 자신도 파멸 |

#### 싱크로 신호 (이런 모습이면 Lv1)

- [ ] 결과를 생각 안 함
- [ ] 가까운 사람도 배신할 수 있음
- [ ] "재미있으면 됐어"
- [ ] 신뢰받지 못함
- [ ] 장난이 파괴가 됨

---

### Lv2: 각성 (Awakening) — "신들의 동반자"

#### 특징

```
"나는 그들과 다르다. 하지만 함께하기도 한다."
```

- **행동 패턴**: 선택적 협력, 도움과 장난
- **대인 관계**: 복잡하지만 연결됨
- **에너지 상태**: 불안정하지만 방향 있음
- **자기 인식**: "나는 필요한 존재다"

#### 신화적 대응

> **토르와 함께하는 로키**
>
> 로키는 토르와 여러 모험을 함께합니다.
> 그의 지혜가 토르의 힘을 보완합니다.
> 스카디를 웃게 만들고, 스바딜파리를 유인하고,
> 신들을 위해 일하기도 합니다.
>
> **Lv2의 로키는 파괴만이 아닌 협력도 하는 상태입니다.**

#### 싱크로 신호 (이런 모습이면 Lv2)

- [ ] 때로는 도움이 됨
- [ ] 복잡한 관계가 있음
- [ ] 완전한 배신자는 아님
- [ ] 자신의 가치를 앎
- [ ] 하지만 여전히 예측 불가

---

### Lv3: 통합 (Integration) — "변화의 촉매"

#### 특징

```
"나는 변화를 일으킨다. 필요한 변화를."
```

- **행동 패턴**: 목적 있는 혼돈, 건설적 파괴
- **대인 관계**: 필요할 때 신뢰받음
- **에너지 상태**: 통제된 혼돈
- **자기 인식**: "나는 변화의 촉매다"

#### 신화적 대응

> **신들의 보물을 가져온 로키**
>
> 로키는 시프의 머리카락을 잘랐다가,
> 대가로 신들에게 보물을 가져옵니다.
> 토르의 묠니르, 오딘의 궁니르, 프레이의 배.
> 파괴가 창조가 되는 순간입니다.
>
> **Lv3의 로키는 혼돈을 통해 새로운 것을 만드는 상태입니다.**

#### 싱크로 신호 (이런 모습이면 Lv3)

- [ ] 파괴 후 창조가 따라옴
- [ ] 변화가 필요할 때 일으킴
- [ ] 결과를 고려함
- [ ] 때로는 신뢰받음
- [ ] 혼돈에 목적이 있음

---

### Lv4: 초월 (Transcendence) — "라그나로크의 촉발자"

#### 특징

```
"끝이 있어야 새로운 시작이 있다."
```

- **행동 패턴**: 우주적 변화의 도구
- **대인 관계**: 적이자 필연
- **에너지 상태**: 운명적 힘
- **자기 인식**: "나는 끝과 시작을 잇는다"

#### 신화적 대응

> **라그나로크의 로키**
>
> 라그나로크에서 로키는 거인들의 편에 섭니다.
> 헤임달과 싸워 함께 죽습니다.
> 하지만 라그나로크가 끝나면 새 세계가 옵니다.
> 로키의 파괴는 새로운 창조의 조건이 됩니다.
>
> **Lv4의 로키는 종말과 시작을 연결하는 존재입니다.**

#### 도달 경지

| 경지 | 발현 |
|------|------|
| 우주적 역할 | 개인을 넘어선 의미 |
| 필연적 혼돈 | 세상에 필요한 변화 |
| 죽음과 재생 | 끝이 곧 시작 |
| 역설적 존재 | 적이자 필연 |

## 신화 인물 3: 프로메테우스 (Prometheus as Rebel)

| **역할** | 제우스에 반항한 티탄 |

### 동기 프로파일

### 핵심 특징 3가지

1. **신에 대한 반역** — 제우스의 명령을 거역
2. **인류를 위한 희생** — 자신의 고통을 감수

---

### Lv1: 그림자 (Shadow) — "무모한 도전자"

#### 특징

```
"신이라도 틀릴 수 있다. 내가 바로잡겠다."
```

- **행동 패턴**: 무모한 도전, 결과 무시
- **대인 관계**: 독단적 결정

| 인물 | 반항 대상 | 반항 방식 | 약점 | 초월 방식 |
|------|-----------|-----------|------|-----------|
| **손오공** | 천상 | 무력 | 무목적 | 성불 |
| **로키** | 질서 | 장난/트릭 | 배신 | 종말과 시작 |

### 레벨별 핵심 과제

| 레벨 | 손오공 | 로키 | 프로메테우스 | 세트 | 마우이 | 루시퍼 |
|------|--------|------|-------------|------|--------|--------|
| **Lv1→2** | 목적 찾기 | 협력 시작 | 누구를 위해 | 영역 인식 | 방향 찾기 | 영역 개척 |
| **Lv2→3** | 지킬 것 | 건설적 혼돈 | 대가 감당 | 필요한 힘 | 불가능 도전 | 자유 제공 |
| **Lv3→4** | 내면 자유 | 우주적 역할 | 순환 해방 | 균형 | 죽음 초월 | 본질 귀환 |

### 당신에게 맞는 인물은?

```
Freedom이 높고...

+ Achievement가 높다 → 손오공, 마우이
+ Creation이 높다 → 로키
+ Connection이 높다 → 프로메테우스
+ Recognition이 높다 → 루시퍼, 세트
+ Adventure가 높다 → 마우이, 로키
```

## 부록: 레벨별 핵심 질문

### 자가 진단 질문

**Lv1 확인**
- 규칙이면 무조건 반대하는가?
- 왜 반항하는지 설명 못하는가?
- 파괴 후 공허함을 느끼는가?

**Lv2 확인**
- 무엇에 반항하는지 아는가?
- 목적이 생겼는가?
- 선택적으로 규칙을 따를 수 있는가?

**Lv3 확인**
- 무엇을 위해 싸우는지 명확한가?
- 지키고 싶은 것이 있는가?
- 규칙도 필요할 때가 있음을 아는가?

**Lv4 확인**
- 규칙에 반응하지 않고 자유로운가?
- 존재 자체가 자유인가?
- 반항이 아닌 해방을 주는가?
