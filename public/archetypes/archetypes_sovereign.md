## 원형 개요

### 핵심 정체성

**The Sovereign (지배자)**는 **영향력과 인정**에서 존재 의미를 찾는 원형입니다.

```
"중심에 서지 않으면 존재하지 않는 것과 같다."
```

### 동기 가중치

| 동기 | 가중치 | 역할 |
|------|--------|------|
| **Recognition** | 0.60 | 핵심 동기 |
| **Achievement** | 0.25 | 보조 동기 |
| **Security** | 0.15 | 보조 동기 |

### 원형 일치 조건

```typescript
conditions: {
  recognition >= 65,
  recognition > connection,
  achievement >= 50
}
```

### 4단계 성숙도 구조

| 레벨 | 이름 | 핵심 | 신화적 단계 |
|------|------|------|-------------|
| **Lv1** | 그림자 (Shadow) | 권력 집착, 폭군, 불안한 왕좌 | 참칭/찬탈 |
| **Lv2** | 각성 (Awakening) | 영향력 인식, 리더십 발현 | 왕위 계승 |
| **Lv3** | 통합 (Integration) | 책임지는 왕, 섬기는 지도자 | 성군/명군 |
| **Lv4** | 초월 (Transcendence) | 왕좌를 넘어선 지혜, 유산 | 신화적 왕 |

| **이름** | Zeus (제우스) |
| **신화 체계** | 그리스 |
| **역할** | 신들의 왕, 하늘과 천둥의 신 |
| **상징** | 번개, 독수리, 왕홀, 참나무 |

### 동기 프로파일

```
Recognition:  0.85  █████████████████ (핵심)
```

### 핵심 특징 3가지

1. **최고의 자리** — 올림포스의 절대 권력
2. **질서의 수호** — 신들과 인간 세계의 질서 유지
3. **번개의 분노** — 도전에 대한 가혹한 응징

### Lv1: 그림자 (Shadow) — "아버지를 삼킨 자의 불안"

#### 특징

```
"아무도 내 자리를 넘볼 수 없다. 의심되면 제거한다."
```

- **행동 패턴**: 잠재적 위협 제거, 편집증적 통제
- **대인 관계**: 충성을 의심, 감시와 처벌
- **에너지 상태**: 불안에서 오는 과잉 경계
- **자기 인식**: "나는 왕이다. 도전은 반역이다"

#### 신화적 대응

> **크로노스의 아들, 크로노스의 두려움**
>
> 제우스는 아버지 크로노스를 무너뜨렸습니다.
> 크로노스도 자신의 아버지 우라노스를 무너뜨렸습니다.
> 제우스는 알고 있습니다 — 언젠가 자신도 그렇게 될 수 있다고.
> 
> 그래서 그는 예언을 두려워합니다.
> 메티스를 삼킨 것도 그녀의 아들이 자신을 넘어설 것이라는 예언 때문.
>
> **Lv1의 제우스는 왕좌를 잃을 두려움에 사로잡힌 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 편집증 | 모든 것을 위협으로 인식 |
| 과잉 통제 | 자율성 억압 |
| 폭력적 응징 | 사소한 도전에도 과잉 반응 |
| 고립 | 진정한 신뢰 관계 없음 |

#### 싱크로 신호 (이런 모습이면 Lv1)

- [ ] 비판을 "반역"으로 느낌
- [ ] 권위에 대한 도전을 참지 못함
- [ ] 주변 사람들을 의심함
- [ ] 통제하지 못하면 불안
- [ ] 자리를 잃을까봐 두려움

---

### Lv2: 각성 (Awakening) — "타이탄을 무너뜨린 자"

#### 특징

```
"나는 이 자리를 얻기 위해 싸웠다. 지킬 것이다."
```

- **행동 패턴**: 권력의 정당성 확립, 질서 구축
- **대인 관계**: 동맹 형성, 역할 분배
- **에너지 상태**: 목표 있을 때 강력
- **자기 인식**: "나는 승리한 왕이다"

#### 신화적 대응

> **티타노마키아의 승자**
>
> 제우스는 형제들(포세이돈, 하데스)과 함께 타이탄들과 싸웠습니다.
> 키클롭스에게 번개를, 헤카톤케이레스에게 도움을 받았습니다.
> 승리 후 세계를 나눕니다:
> 제우스는 하늘, 포세이돈은 바다, 하데스는 지하.
>
> **Lv2의 제우스는 동맹을 통해 권력을 확립한 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 승자의 오만 | "내가 이겼으니 옳다" |
| 분배의 불균형 | 자신에게 유리하게 |
| 과거 의존 | 과거 공적으로 정당화 |
| 새 위협 경시 | 이미 이겼다는 안일함 |

#### 싱크로 신호 (이런 모습이면 Lv2)

- [ ] 과거 성취로 권위 주장
- [ ] 동맹/팀을 구축함
- [ ] 역할 분배를 함 (자신에게 유리하게)
- [ ] 승리에 대한 자부심
- [ ] 새로운 도전에는 여전히 민감

---

### Lv3: 통합 (Integration) — "올림포스의 질서"

#### 특징

```
"왕의 역할은 질서를 지키는 것이다."
```

- **행동 패턴**: 질서 유지, 중재, 정의 실현
- **대인 관계**: 공정한 판결, 신뢰 구축
- **에너지 상태**: 안정적 통치
- **자기 인식**: "나는 질서의 수호자"

#### 신화적 대응

> **제노도토스(손님의 수호자) 제우스**
>
> 제우스는 손님을 대접하는 법(환대)의 수호자입니다.
> 신들 사이의 분쟁을 중재합니다.
> 인간 세계의 정의를 감시합니다.
> 폭군이 아닌 질서의 왕입니다.
>
> **Lv3의 제우스는 권력을 질서와 정의에 사용하는 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 이중 잣대 | 자신에게는 관대 |
| 정의의 과잉 | 자비 없는 판결 |
| 질서 집착 | 예외를 인정 못함 |

#### 싱크로 신호 (이런 모습이면 Lv3)

- [ ] 공정하게 판단하려 노력
- [ ] 갈등을 중재함
- [ ] 질서와 규칙을 중시
- [ ] 권력을 책임으로 인식
- [ ] 약자를 보호하려 함

---

### Lv4: 초월 (Transcendence) — "운명보다 높은 왕"

#### 특징

```
"왕좌는 수단이다. 목적은 세상의 질서다."
```

- **행동 패턴**: 왕좌를 넘어선 통치
- **대인 관계**: 적까지 포용
- **에너지 상태**: 고요한 권위
- **자기 인식**: "나는 왕 이상이다"

#### 신화적 대응

> **운명(모이라이)과 제우스**
>
> 제우스조차 운명을 거스를 수 없다고 합니다.
> 하지만 그는 운명과 협력합니다.
> 저항하지 않고 받아들이며 최선을 만듭니다.
> 
> 트로이 전쟁에서 아들 사르페돈이 죽을 운명이었을 때,
> 제우스는 구하고 싶었지만 운명을 받아들였습니다.
>
> **Lv4의 제우스는 자신의 한계조차 받아들이는 상태입니다.**

#### 도달 경지

| 경지 | 발현 |
|------|------|
| 한계 수용 | 운명도 거스르지 않음 |
| 자녀 수용 | 아테나(메티스의 딸)도 받아들임 |
| 왕좌 초월 | 자리보다 역할 |
| 영원한 질서 | 올림포스는 영원 |

#### 싱크로 신호 (이런 모습이면 Lv4)

- [ ] 자신의 한계를 인정함
- [ ] 권력보다 역할에 집중
- [ ] 후계자를 기꺼이 인정
- [ ] 적의 가치도 인정
- [ ] 왕좌 없이도 권위 있음

---

### 레벨 진화 조건

```
Lv1 → Lv2: 두려움에서 벗어나 동맹을 구축할 때
Lv2 → Lv3: 권력을 질서와 정의에 사용할 때
Lv3 → Lv4: 자신의 한계를 받아들이고 왕좌를 초월할 때
```

### 레벨 퇴화 경고

```
Lv4 → Lv3: 다시 권력에 집착할 때
Lv3 → Lv2: 정의보다 승리를 우선할 때
Lv2 → Lv1: 동맹을 의심하고 고립될 때
```

## 신화 인물 2: 옥황상제 (玉皇上帝)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | 玉皇上帝 (옥황상제, Jade Emperor) |
| **신화 체계** | 중국 도교 |
| **역할** | 천상의 황제, 삼계의 지배자 |
| **상징** | 용포, 면류관, 옥새, 구름 |

### 동기 프로파일

```
Recognition:  0.80  ████████████████ (핵심)
Security:     0.70  ██████████████
Achievement:  0.60  ████████████
Connection:   0.50  ██████████
Mastery:      0.45  █████████
Freedom:      0.30  ██████
```

### 핵심 특징 3가지

1. **최고의 권위** — 모든 신들 위에 군림
2. **관료제 통치** — 천상의 체계적 관료 시스템
3. **덕으로 다스림** — 힘보다 덕과 질서

---

### Lv1: 그림자 (Shadow) — "관료제에 갇힌 황제"

#### 특징

```
"절차대로 해라. 예외는 없다."
```

- **행동 패턴**: 형식에 집착, 유연성 없음
- **대인 관계**: 위계에 의한 관계만
- **에너지 상태**: 안정적이지만 경직
- **자기 인식**: "나는 체계 그 자체다"

#### 신화적 대응

> **손오공에게 당한 옥황상제**
>
> 서유기에서 손오공은 천상을 뒤집어 놓습니다.
> 옥황상제는 관료제에 의존하여 대응합니다.
> "누구를 보내라", "어떤 절차로 처리하라"
> 직접 나서지 않고, 체계에만 의존합니다.
> 결국 부처에게 도움을 청해야 했습니다.
>
> **Lv1의 옥황상제는 체계에 갇혀 유연성을 잃은 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 형식주의 | 절차가 목적이 됨 |
| 위임 과잉 | 직접 나서지 않음 |
| 경직 | 예외 상황 대처 불가 |
| 권위 위탁 | 실제 힘은 남에게 |

#### 싱크로 신호 (이런 모습이면 Lv1)

- [ ] 절차와 형식에 집착
- [ ] 직접 나서기를 피함
- [ ] 위기 시 대응이 느림
- [ ] 예외를 인정하지 않음
- [ ] 권위는 있지만 영향력은 약함

---

### Lv2: 각성 (Awakening) — "천상의 질서를 세운 자"

#### 특징

```
"체계가 있어야 세상이 돌아간다."
```

- **행동 패턴**: 효율적 시스템 구축
- **대인 관계**: 역할과 책임 명확화
- **에너지 상태**: 체계 안에서 안정
- **자기 인식**: "나는 시스템의 설계자"

#### 신화적 대응

> **천상의 관료제를 세운 옥황상제**
>
> 옥황상제는 천상에 정교한 관료 시스템을 만듭니다.
> 사해용왕, 염라대왕, 토지신, 문창제군...
> 각자의 역할과 책임이 명확합니다.
> 인간 세상의 관료제 모델이 됩니다.
>
> **Lv2의 옥황상제는 체계를 통해 통치하는 상태입니다.**

#### 싱크로 신호 (이런 모습이면 Lv2)

- [ ] 시스템을 만들고 운영함
- [ ] 역할 분배가 명확
- [ ] 체계적으로 일이 돌아감
- [ ] 직접 나서기보다 위임
- [ ] 예외 상황에는 여전히 약함

---

### Lv3: 통합 (Integration) — "덕으로 다스리는 황제"

#### 특징

```
"힘이 아닌 덕으로, 체계가 아닌 마음으로."
```

- **행동 패턴**: 덕치(德治), 유연한 판단
- **대인 관계**: 신뢰와 존경 기반
- **에너지 상태**: 안정적이면서 유연
- **자기 인식**: "나는 덕의 화신"

#### 신화적 대응

> **중생을 구제하는 옥황상제**
>
> 도교 전통에서 옥황상제는 자비로운 존재입니다.
> 인간의 고통을 듣고 응답합니다.
> 체계만이 아닌, 마음으로 다스립니다.
> 절차를 넘어 정의를 실현합니다.
>
> **Lv3의 옥황상제는 체계와 덕을 함께 사용하는 상태입니다.**

#### 싱크로 신호 (이런 모습이면 Lv3)

- [ ] 절차보다 목적을 중시
- [ ] 예외를 인정할 수 있음
- [ ] 부하들의 신뢰를 받음
- [ ] 위기 시 직접 나설 수 있음
- [ ] 권위와 실력이 일치

---

### Lv4: 초월 (Transcendence) — "도(道)와 하나 된 황제"

#### 특징

```
"다스리되 다스리지 않는다. 무위이치(無爲而治)."
```

- **행동 패턴**: 무위의 통치, 자연스러운 질서
- **대인 관계**: 존재 자체가 질서
- **에너지 상태**: 고요한 권위
- **자기 인식**: "나는 도의 발현"

#### 신화적 대응

> **도와 하나 된 옥황상제**
>
> 도교의 최고 경지는 무위(無爲)입니다.
> 아무것도 하지 않으면서 모든 것이 이루어지는 상태.
> 옥황상제가 이 경지에 이르면,
> 천상은 그가 다스리지 않아도 스스로 질서를 유지합니다.
>
> **Lv4의 옥황상제는 무위로 다스리는 상태입니다.**

#### 도달 경지

| 경지 | 발현 |
|------|------|
| 무위이치 | 다스리지 않고 다스림 |
| 체계 초월 | 시스템 없이도 질서 |
| 도와 일치 | 존재가 곧 법칙 |
| 영원한 질서 | 황제 없이도 천상은 유지 |

3. **쇠퇴하는 위상** — 후기 힌두교에서 위상 하락

### Lv1: 그림자 (Shadow) — "왕좌에 집착하는 신"

#### 특징

```
"이 자리는 내가 피로 얻었다. 아무도 뺏을 수 없다."
```

- **행동 패턴**: 잠재적 도전자 제거
- **대인 관계**: 의심과 방해
- **에너지 상태**: 불안정, 경계
- **자기 인식**: "나는 왕이다 (계속 왕이어야 한다)"

#### 신화적 대응

> **수행자를 방해하는 인드라**
>
> 인드라는 자신의 자리를 위협할 수 있는
> 강력한 수행자들의 수행을 방해합니다.
> 압사라를 보내 유혹하거나, 시련을 내립니다.
> 불안이 방해로 나타납니다.
>
> **Lv1의 인드라(왕)는 도전에 대한 불안으로 방해하는 상태입니다.**

#### 싱크로 신호 (이런 모습이면 Lv1)

- [ ] 잠재적 경쟁자가 두려움
- [ ] 성장하는 후배를 견제
- [ ] 왕좌를 잃을까봐 불안
- [ ] 방해와 견제로 지위 유지
- [ ] 불안이 사라지지 않음

### Lv2: 각성 (Awakening) — "브리트라를 죽인 영웅"

#### 특징

```
"나는 악을 물리쳐 이 자리를 얻었다."
```

- **행동 패턴**: 업적으로 정당성 확립
- **대인 관계**: 영웅으로서의 존경
- **에너지 상태**: 승리의 기억으로 버팀
- **자기 인식**: "나는 영웅이자 왕"

#### 신화적 대응

> **브리트라(Vritra)를 죽인 인드라**
>
> 인드라는 혼돈의 용 브리트라를 죽였습니다.
> 브리트라는 물을 가두어 세상에 가뭄을 일으켰습니다.
> 인드라가 그를 죽이자 물이 흘러 세상이 살아났습니다.
> 이 영웅적 행위로 그는 왕이 되었습니다.
>
> **Lv2의 인드라(왕)는 과거 업적으로 정당성을 주장하는 상태입니다.**

---

### Lv3: 통합 (Integration) — "비를 내리는 왕"

#### 특징

```
"왕의 역할은 백성을 돌보는 것이다."
```

- **행동 패턴**: 통치 역할 수행, 비를 내림
- **대인 관계**: 백성을 위한 봉사
- **에너지 상태**: 안정적 통치
- **자기 인식**: "나는 풍요의 원천"

#### 신화적 대응

> **비를 내리는 인드라**
>
> 인드라는 비의 신이기도 합니다.
> 농부들이 기도하면 비를 내립니다.
> 전쟁의 신에서 풍요의 신으로.
> 싸우는 왕에서 주는 왕으로.
>
> **Lv3의 인드라(왕)는 백성을 위해 베푸는 상태입니다.**

---

### Lv4: 초월 (Transcendence) — "겸손한 왕"

#### 특징

```
"왕좌보다 역할이 중요하다."
```

- **행동 패턴**: 지위보다 역할
- **대인 관계**: 더 높은 존재(비슈누, 시바) 인정
- **에너지 상태**: 겸손한 권위
- **자기 인식**: "나는 나의 역할을 다한다"

#### 신화적 대응

> **비슈누와 시바 앞에서 겸손한 인드라**
>
> 힌두교가 발전하면서 인드라의 위상은 낮아집니다.
> 비슈누, 시바, 브라흐마가 더 높이 올라갑니다.
> 하지만 인드라는 여전히 자신의 역할을 합니다.
> 왕좌가 낮아져도 역할은 변하지 않습니다.
>
> **Lv4의 인드라(왕)는 겸손하게 역할을 다하는 상태입니다.**
