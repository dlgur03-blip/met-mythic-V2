## 원형 개요

### 핵심 정체성

**The Healer (치유자)**는 **연결과 기여**에서 존재 의미를 찾는 원형입니다.

```
"누군가를 돕지 않으면 내 존재 이유를 모르겠다."
```

### 동기 가중치

| 동기 | 가중치 | 역할 |
|------|--------|------|
| **Connection** | 0.70 | 핵심 동기 |
| **Security** | 0.15 | 보조 동기 |
| **Creation** | 0.15 | 보조 동기 |

### 원형 일치 조건

```typescript
conditions: {
  connection >= 70,
  connection > recognition,
  security >= 40
}
```

### 4단계 성숙도 구조

| 레벨 | 이름 | 핵심 | 신화적 단계 |
|------|------|------|-------------|
| **Lv1** | 그림자 (Shadow) | 공의존, 자기희생, 경계 없음 | 상처받은 치유자 |
| **Lv2** | 각성 (Awakening) | 돌봄의 기쁨, 연결 시작 | 치유 시작 |
| **Lv3** | 통합 (Integration) | 건강한 돌봄, 경계와 연민 | 온전한 치유자 |
| **Lv4** | 초월 (Transcendence) | 무조건적 사랑, 존재가 치유 | 신성한 치유자 |

## 신화 인물 1: 관음 (觀音, Guanyin)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | 觀音/觀世音 (관음/관세음, Guanyin) |
| **신화 체계** | 불교 (동아시아) |
| **역할** | 자비의 보살, 고통받는 자의 구원자 |
| **상징** | 연꽃, 정병(淨甁), 천 개의 손, 흰 옷 |

### 동기 프로파일

```
Connection:  0.95  ███████████████████ (핵심)
Security:    0.50  ██████████
Creation:    0.40  ████████
Mastery:     0.45  █████████
Freedom:     0.30  ██████
Recognition: 0.20  ████
```

### 핵심 특징 3가지

1. **무한한 자비** — 모든 고통을 듣고 응답
2. **천 개의 손** — 모든 이를 동시에 도움
3. **성불 거부** — 모든 중생이 구원받을 때까지 기다림

---

### Lv1: 그림자 (Shadow) — "모든 고통을 짊어지려는 자"

#### 특징

```
"모든 고통을 내가 대신 짊어지겠다."
```

- **행동 패턴**: 자기 희생 과잉, 경계 없음
- **대인 관계**: 공의존, 필요받아야 존재감
- **에너지 상태**: 고갈, 번아웃
- **자기 인식**: "남을 돕지 않으면 가치 없다"

#### 신화적 대응

> **머리가 터진 관음**
>
> 관음은 모든 중생의 고통을 듣겠다고 서원했습니다.
> 하지만 고통의 소리가 너무 많아
> 그의 머리가 터져버렸습니다.
> 아미타불이 11개의 머리를 주어 다시 들을 수 있게 했습니다.
>
> **Lv1의 관음은 감당할 수 없는 고통을 짊어지려다 무너지는 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 과잉 희생 | 자신을 돌보지 않음 |
| 경계 없음 | 모든 요청에 "네" |
| 구원자 콤플렉스 | "나만이 도울 수 있다" |
| 공의존 | 필요받아야 존재감 |

#### 싱크로 신호 (이런 모습이면 Lv1)

- [ ] 거절하지 못함
- [ ] 남을 돕느라 자신은 지침
- [ ] 도움받는 것이 불편함
- [ ] "내가 없으면 안 돼"라는 생각
- [ ] 돕지 않으면 죄책감

---

### Lv2: 각성 (Awakening) — "천수천안(千手千眼)"

#### 특징

```
"내가 할 수 있는 만큼 돕는다."
```

- **행동 패턴**: 지속 가능한 돌봄, 능력 확장
- **대인 관계**: 건강한 연결 시작
- **에너지 상태**: 돌봄에서 에너지 얻음
- **자기 인식**: "나는 도울 수 있다"

#### 신화적 대응

> **천 개의 손과 천 개의 눈**
>
> 관음은 더 많은 이를 돕기 위해
> 천 개의 손과 천 개의 눈을 얻습니다.
> 더 많이 보고, 더 많이 닿을 수 있게.
> 하지만 이것은 능력의 확장이지, 무한한 희생이 아닙니다.
>
> **Lv2의 관음은 능력을 키워 더 많이 돕는 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 능력 과신 | 천 개의 손도 한계가 있음 |
| 양적 집착 | 더 많이, 더 많이 |
| 자기 돌봄 부족 | 여전히 자신은 뒷전 |
| 피로 누적 | 지속 불가능한 속도 |

#### 싱크로 신호 (이런 모습이면 Lv2)

- [ ] 돕는 능력을 키움
- [ ] 더 많은 사람을 도움
- [ ] 돕는 것에서 기쁨을 느낌
- [ ] 하지만 여전히 지침
- [ ] 거절이 어려움

---

### Lv3: 통합 (Integration) — "자비와 지혜의 균형"

#### 특징

```
"자비에는 지혜가 필요하다."
```

- **행동 패턴**: 경계 있는 자비, 지혜로운 돌봄
- **대인 관계**: 건강한 주고받음
- **에너지 상태**: 지속 가능한 돌봄
- **자기 인식**: "돌봄에도 지혜가 필요하다"

#### 신화적 대응

> **반야심경을 가르치는 관음**
>
> 관음은 사리불에게 반야심경을 가르칩니다.
> "색즉시공 공즉시색"
> 자비만이 아닌 지혜(반야)도 함께.
> 감정만이 아닌 통찰도 필요합니다.
>
> **Lv3의 관음은 자비와 지혜를 함께 사용하는 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 거리두기 과잉 | 지혜가 냉정함이 되기도 |
| 판단 | "이 사람은 도울 가치가..." |
| 선별적 자비 | 조건부 돌봄 |

#### 싱크로 신호 (이런 모습이면 Lv3)

- [ ] 건강하게 거절할 수 있음
- [ ] 자신도 돌봄
- [ ] 지혜롭게 돕는 방법을 선택
- [ ] 주는 것과 받는 것의 균형
- [ ] 지속 가능한 돌봄

---

### Lv4: 초월 (Transcendence) — "성불을 미루는 보살"

#### 특징

```
"모든 중생이 구원받을 때까지 나는 기다린다."
```

- **행동 패턴**: 무조건적 자비, 존재 자체가 치유
- **대인 관계**: 모든 존재와 연결
- **에너지 상태**: 무한한 자비의 샘
- **자기 인식**: "나는 자비 그 자체"

#### 신화적 대응

> **성불을 거부한 관음**
>
> 관음은 이미 성불할 수 있었습니다.
> 하지만 그는 서원했습니다:
> "지옥이 비지 않으면 성불하지 않겠다.
> 모든 중생이 구원받을 때까지 기다리겠다."
>
> 이것은 희생이 아닌 사랑입니다.
> 조건 없는, 기대 없는, 순수한 자비.
>
> **Lv4의 관음은 무조건적 사랑 그 자체인 상태입니다.**

#### 도달 경지

| 경지 | 발현 |
|------|------|
| 무조건적 자비 | 조건/기대 없이 |
| 성불 초월 | 자신의 구원보다 타인 |
| 존재가 치유 | 있는 것만으로 |
| 영원한 서원 | 끝없이 계속 |

#### 싱크로 신호 (이런 모습이면 Lv4)

- [ ] 보답을 기대하지 않음
- [ ] 존재 자체가 평화로움
- [ ] 모든 존재에 연민
- [ ] 돕지 않아도 연결됨
- [ ] 자비가 자연스러움

---

### 레벨 진화 조건

```
Lv1 → Lv2: 자기 희생에서 지속 가능한 돌봄으로
Lv2 → Lv3: 자비에 지혜를 더할 때
Lv3 → Lv4: 조건 없는 사랑에 도달할 때
```

### 레벨 퇴화 경고

```
Lv4 → Lv3: 다시 조건을 붙이기 시작할 때
Lv3 → Lv2: 경계가 무너지고 과잉 돌봄에 빠질 때
Lv2 → Lv1: 자기 돌봄을 멈추고 번아웃될 때
```

## 신화 인물 2: 헤스티아 (Hestia)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | Hestia (헤스티아) |
| **신화 체계** | 그리스 |
| **역할** | 화덕의 여신, 가정의 수호신 |
| **상징** | 화덕, 불꽃, 원 |

### 동기 프로파일

```
Connection:  0.80  ████████████████ (핵심)
Security:    0.85  █████████████████
Creation:    0.30  ██████
Freedom:     0.25  █████
Recognition: 0.15  ███
Achievement: 0.20  ████
```

### 핵심 특징 3가지

1. **조용한 헌신** — 주목받지 않고 돌봄
2. **화덕의 불** — 가정의 중심, 따뜻함의 원천
3. **순결한 평화** — 갈등을 피하고 조화 추구

---

### Lv1: 그림자 (Shadow) — "보이지 않는 존재"

#### 특징

```
"나는 여기 있지만 아무도 나를 보지 않는다."
```

- **행동 패턴**: 자기 지움, 투명인간
- **대인 관계**: 존재감 없음, 무시당함
- **에너지 상태**: 고요하지만 공허함
- **자기 인식**: "나는 중요하지 않다"

#### 신화적 대응

> **올림포스 12신에서 빠진 헤스티아**
>
> 헤스티아는 원래 올림포스 12신 중 하나였습니다.
> 하지만 디오니소스가 올라올 때 자리를 양보했습니다.
> 갈등 없이, 조용히.
> 그녀는 여전히 모든 신전의 화덕에 있지만,
> 누구도 그녀를 기억하지 않습니다.
>
> **Lv1의 헤스티아는 자신을 지워버린 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 자기 지움 | 존재감을 스스로 없앰 |
| 과잉 양보 | 자기 자리를 내줌 |
| 무시 수용 | 무시당하는 것을 당연시 |
| 욕구 부정 | "나는 필요 없어" |

#### 싱크로 신호 (이런 모습이면 Lv1)

- [ ] 자신을 드러내지 않음
- [ ] 무시당해도 괜찮다고 함
- [ ] 자기 의견을 말하지 않음
- [ ] 갈등을 피하려고 양보
- [ ] "나는 중요하지 않아"

---

### Lv2: 각성 (Awakening) — "화덕을 지키는 자"

#### 특징

```
"나는 이 불을 지킨다. 그것이 나의 역할이다."
```

- **행동 패턴**: 역할 인식, 조용한 헌신
- **대인 관계**: 뒤에서 지원
- **에너지 상태**: 안정적, 따뜻함
- **자기 인식**: "나는 화덕의 수호자"

#### 신화적 대응

> **모든 신전의 화덕에 있는 헤스티아**
>
> 헤스티아의 불은 모든 신전에 있습니다.
> 제우스의 신전에도, 아테나의 신전에도.
> 모든 가정의 화덕에도.
> 보이지 않지만 모든 곳에 있습니다.
>
> **Lv2의 헤스티아는 자신의 역할을 인식한 상태입니다.**

#### 싱크로 신호 (이런 모습이면 Lv2)

- [ ] 자신의 역할을 알고 있음
- [ ] 조용히 헌신함
- [ ] 인정받지 못해도 역할 수행
- [ ] 안정감을 제공함
- [ ] 하지만 여전히 자기 욕구는 숨김

---

### Lv3: 통합 (Integration) — "가정의 중심"

#### 특징

```
"화덕이 있어야 가정이 있다. 나는 중요하다."
```

- **행동 패턴**: 자기 가치 인정, 건강한 돌봄
- **대인 관계**: 중심으로서 연결
- **에너지 상태**: 지속적 따뜻함
- **자기 인식**: "나는 중심이다"

#### 신화적 대응

> **첫 번째이자 마지막인 헤스티아**
>
> 모든 제사에서 헤스티아는 첫 번째와 마지막으로 불립니다.
> 시작과 끝, 그녀가 있어야 의식이 완성됩니다.
> 보이지 않지만 없어서는 안 됩니다.
> 그녀는 이것을 압니다.
>
> **Lv3의 헤스티아는 자신의 중요성을 아는 상태입니다.**

#### 싱크로 신호 (이런 모습이면 Lv3)

- [ ] 자신의 가치를 인정함
- [ ] 필요할 때 목소리를 냄
- [ ] 주목받지 않아도 괜찮지만, 무시당하면 안 됨을 앎
- [ ] 돌봄과 자기 존중의 균형
- [ ] 중심으로서 연결을 만듦

---

### Lv4: 초월 (Transcendence) — "영원한 불꽃"

#### 특징

```
"나는 불꽃이다. 꺼지지 않는다."
```

- **행동 패턴**: 존재 자체가 따뜻함
- **대인 관계**: 모든 가정의 중심
- **에너지 상태**: 영원한 불꽃
- **자기 인식**: "나는 화덕 그 자체"

#### 신화적 대응

> **꺼지지 않는 불꽃**
>
> 헤스티아의 불은 절대 꺼지지 않습니다.
> 로마에서 베스타의 처녀들이 불을 지켰습니다.
> 불이 꺼지면 로마가 망한다고 믿었습니다.
> 그녀는 문명의 영속성 그 자체입니다.
>
> **Lv4의 헤스티아는 영원한 따뜻함의 원천인 상태입니다.**

### Lv1: 그림자 (Shadow) — "집착하는 사랑"

#### 특징

```
"당신 없이 나는 살 수 없다."
```

- **행동 패턴**: 집착, 의존
- **대인 관계**: 상대에게 매달림
- **에너지 상태**: 상대에 의해 결정됨
- **자기 인식**: "당신이 나의 전부"

#### 신화적 대응

> **세 명의 브리지드**
>
> 브리지드는 세 여신의 이름입니다.
> 치유의 브리지드, 시의 브리지드, 대장장이의 브리지드.
> 셋인가, 하나인가?
> Lv1에서는 이 분열이 정체성 혼란이 됩니다.
>
> **Lv1의 브리지드는 자신이 누구인지 모르는 상태입니다.**

#### 싱크로 신호 (이런 모습이면 Lv1)

- [ ] 여러 역할 사이에서 혼란
- [ ] 무엇을 해야 할지 모름
- [ ] 에너지가 분산됨
- [ ] 하나에 집중하지 못함
- [ ] "나는 누구인가?"

### Lv2: 각성 (Awakening) — "불을 지키는 자"

#### 특징

```
"나의 불이 세상을 비춘다."
```

- **행동 패턴**: 하나의 역할에 집중
- **대인 관계**: 따뜻함 제공
- **에너지 상태**: 집중된 힘
- **자기 인식**: "나는 불의 수호자"

#### 신화적 대응

> **킬데어의 영원한 불**
>
> 브리지드의 신전에는 꺼지지 않는 불이 있습니다.
> 19명의 여사제가 돌아가며 불을 지킵니다.
> 20일째는 브리지드 자신이 지킵니다.
> 불에 집중함으로써 정체성을 찾습니다.
>
> **Lv2의 브리지드는 하나의 역할에서 정체성을 찾은 상태입니다.**

### Lv3: 통합 (Integration) — "삼위일체의 통합"

#### 특징

```
"치유, 창조, 변형 — 모두 하나다."
```

- **행동 패턴**: 여러 역할의 통합
- **대인 관계**: 다양한 방식으로 돕기
- **에너지 상태**: 통합된 힘
- **자기 인식**: "나는 모든 것이다"

#### 신화적 대응

> **불과 물을 함께 가진 브리지드**
>
> 브리지드는 화덕의 불과 치유의 우물을 함께 가집니다.
> 불은 변형(대장장이), 물은 치유.
> 시는 영감을 불어넣습니다.
> 세 역할이 하나로 통합됩니다.
>
> **Lv3의 브리지드는 모든 역할을 하나로 통합한 상태입니다.**

### Lv4: 초월 (Transcendence) — "여신에서 성인으로"

#### 특징

```
"형태는 변해도 본질은 영원하다."
```

- **행동 패턴**: 시대를 초월한 돌봄
- **대인 관계**: 모든 시대의 사람들과 연결
- **에너지 상태**: 영원한 불꽃
- **자기 인식**: "나는 영원하다"

#### 신화적 대응

> **성 브리지드**
>
> 기독교가 전래된 후, 브리지드는 성 브리지드가 됩니다.
> 여신이 성인으로 변형됩니다.
> 하지만 본질은 같습니다 — 치유, 따뜻함, 돌봄.
> 형태를 바꾸어 영원히 남습니다.
>
> **Lv4의 브리지드는 형태를 초월한 영원한 존재입니다.**
