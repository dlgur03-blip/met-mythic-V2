## 원형 개요

### 핵심 정체성

**The Explorer (탐험가)**는 **발견과 경험**에서 존재 의미를 찾는 원형입니다.

```
"아직 가보지 않은 길이 있다면, 나는 거기 있어야 한다."
```

### 동기 가중치

| 동기 | 가중치 | 역할 |
|------|--------|------|
| **Adventure** | 0.65 | 핵심 동기 |
| **Freedom** | 0.20 | 보조 동기 |
| **Mastery** | 0.15 | 보조 동기 |

### 원형 일치 조건

```typescript
conditions: {
  adventure >= 65,
  adventure > security,
  freedom >= 50
}
```

### 4단계 성숙도 구조

| 레벨 | 이름 | 핵심 | 신화적 단계 |
|------|------|------|-------------|
| **Lv1** | 그림자 (Shadow) | 도피, 정착 불가, 끝없는 방황 | 방랑자 |
| **Lv2** | 각성 (Awakening) | 목적 있는 탐험, 발견의 기쁨 | 탐험가 |
| **Lv3** | 통합 (Integration) | 내면과 외면의 여정, 귀환 | 영웅의 귀환 |
| **Lv4** | 초월 (Transcendence) | 여정 자체가 삶, 길의 지혜 | 영원한 여행자 |

### 기본 정보

|------|------|
| **이름** | Odysseus (오디세우스) |
| **신화 체계** | 그리스 |
| **역할** | 이타카의 왕, 10년의 귀향 |
| **상징** | 목마, 활, 바다, 배 |

### 동기 프로파일

```
Adventure:   0.85  █████████████████ (핵심)
Mastery:     0.80  ████████████████
Achievement: 0.70  ██████████████
```

### 핵심 특징 3가지

1. **기지와 계략** — 힘보다 지혜로 해결
2. **10년의 귀향** — 집으로 가는 가장 긴 여정
3. **호기심의 대가** — 알고 싶어서 위험에 빠짐

---

### Lv1: 그림자 (Shadow) — "집에 가지 못하는 자"

#### 특징

```
"돌아가고 싶지만, 무언가가 나를 계속 끌어당긴다."
```

- **행동 패턴**: 귀환 실패, 호기심에 의한 우회
- **대인 관계**: 가족을 그리워하지만 만나지 못함
- **에너지 상태**: 분열, 갈등
- **자기 인식**: "나는 집에 가야 한다 (하지만 가지 못한다)"

#### 신화적 대응

> **키클롭스 동굴의 오디세우스**
>
> 오디세우스는 키클롭스 폴리페모스의 동굴에 들어갑니다.
> 호기심 때문에.
> 부하들이 만류했지만, 그는 "보고 싶었습니다."
> 결과는 부하들의 죽음과 포세이돈의 저주.
>
> **Lv1의 오디세우스는 호기심이 재앙이 되는 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 호기심 과잉 | 위험을 무시 |
| 귀환 실패 | 집에 가지 못함 |
| 동료 상실 | 부하들이 죽어감 |
| 자기 합리화 | "어쩔 수 없었어" |

#### 싱크로 신호 (이런 모습이면 Lv1)

- [ ] 호기심 때문에 위험에 빠짐
- [ ] 목적지에 도착하지 못함
- [ ] 중요한 것을 잃으면서도 계속 탐험
- [ ] "돌아가야 하는데" 하면서 안 돌아감
- [ ] 귀환보다 모험이 우선

---

### Lv2: 각성 (Awakening) — "귀향을 시작한 자"

#### 특징

```
"집에 가야 한다. 그것이 나의 진정한 목적이다."
```

- **행동 패턴**: 목적 있는 여정, 귀환 의지
- **대인 관계**: 가족을 향한 그리움
- **에너지 상태**: 방향 있는 에너지
- **자기 인식**: "나는 이타카의 왕, 가장이다"

#### 신화적 대응

> **칼립소를 떠나는 오디세우스**
>
> 칼립소는 오디세우스에게 불멸을 제안합니다.
> 그녀와 함께라면 영원히 살 수 있습니다.
> 하지만 오디세우스는 거절합니다.
> "나는 집에 가야 한다."
> 불멸보다 귀환을 선택합니다.
>
> **Lv2의 오디세우스는 귀환의 의지를 확인한 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 지연 | 여전히 우회함 |
| 유혹 | 새로운 모험의 유혹 |
| 피로 | 지침, 포기하고 싶음 |
| 의심 | 정말 돌아갈 수 있을까 |

#### 싱크로 신호 (이런 모습이면 Lv2)

- [ ] 목적지가 명확함
- [ ] 유혹을 거절할 수 있음
- [ ] 그리운 것이 있음
- [ ] 하지만 아직 도착하지 못함
- [ ] 여정이 계속됨

---

### Lv3: 통합 (Integration) — "귀환한 왕"

#### 특징

```
"돌아왔다. 여정이 나를 바꾸었다."
```

- **행동 패턴**: 귀환 완성, 변화된 자아
- **대인 관계**: 가족과 재회, 왕국 회복
- **에너지 상태**: 통합된 정체성
- **자기 인식**: "나는 떠났던 자이자 돌아온 자"

#### 신화적 대응

> **이타카로 돌아온 오디세우스**
>
> 20년 만에 오디세우스는 이타카로 돌아옵니다.
> 구혼자들을 처치하고 왕국을 되찾습니다.
> 페넬로페와 재회합니다.
> 여정이 끝났습니다 — 아니, 완성되었습니다.
>
> **Lv3의 오디세우스는 여정을 완성하고 귀환한 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 적응 어려움 | 집이 낯설음 |
| 과거 집착 | 모험을 그리워함 |
| 관계 회복 | 변한 나, 변한 가족 |

#### 싱크로 신호 (이런 모습이면 Lv3)

- [ ] 목적지에 도착함
- [ ] 여정이 나를 바꾸었음을 앎
- [ ] 소중한 것과 재회
- [ ] 떠남과 돌아옴의 균형
- [ ] 집에서 편안함

---

### Lv4: 초월 (Transcendence) — "다시 떠나는 자"

#### 특징

```
"집은 내 마음에 있다. 어디든 갈 수 있다."
```

- **행동 패턴**: 여정 자체가 삶
- **대인 관계**: 어디서든 연결됨
- **에너지 상태**: 평화로운 움직임
- **자기 인식**: "나는 영원한 여행자"

#### 신화적 대응

> **단테의 오디세우스**
>
> 단테의 신곡에서 오디세우스는 다시 떠납니다.
> 귀환 후에도 가만히 있지 못합니다.
> "세상의 끝을 보고 싶다."
> 헤라클레스의 기둥을 넘어 미지로.
>
> 혹은 테니슨의 율리시스:
> "멈추지 않고 탐구하고, 찾고, 발견하리라."
>
> **Lv4의 오디세우스는 여정 자체가 삶이 된 상태입니다.**

#### 도달 경지

| 경지 | 발현 |
|------|------|
| 영원한 여정 | 멈추지 않음 |
| 내면의 집 | 어디든 집 |
| 평화로운 탐험 | 도피가 아닌 탐구 |
| 지혜 공유 | 여정의 이야기를 전함 |

#### 싱크로 신호 (이런 모습이면 Lv4)

- [ ] 집이 없어도 편안함
- [ ] 어디든 집처럼 느낌
- [ ] 탐험이 도피가 아님
- [ ] 여정의 지혜를 나눔
- [ ] "아직 갈 곳이 있다"

---

### 레벨 진화 조건

```
Lv1 → Lv2: 왜 여행하는지, 어디로 가는지 알게 될 때
Lv2 → Lv3: 목적지에 도착하고 귀환할 때
Lv3 → Lv4: 어디서든 집을 느끼고 다시 떠날 수 있을 때
```

### 레벨 퇴화 경고

```
Lv4 → Lv3: 다시 정착해야 한다는 압박을 느낄 때
Lv3 → Lv2: 귀환 후 적응하지 못할 때
Lv2 → Lv1: 목적을 잃고 방황할 때
```

| **이름** | Gilgamesh (길가메시) |

| **역할** | 우루크의 왕, 불멸을 찾아 떠난 영웅 |
| **상징** | 사자, 삼나무, 불사의 풀 |

### 동기 프로파일

```
Adventure:   0.80  ████████████████ (핵심)

Mastery:     0.65  █████████████

```

### 핵심 특징 3가지

1. **최초의 영웅** — 인류 최초의 서사시 주인공
2. **불멸의 탐구** — 죽음을 거부하는 여정
3. **엔키두의 죽음** — 친구의 상실이 여정의 시작

---

### Lv1: 그림자 (Shadow) — "죽음을 피해 도망치는 자"

#### 특징

```
"나는 죽고 싶지 않다. 영원히 살아야 한다."
```

- **행동 패턴**: 죽음 공포, 도피
- **대인 관계**: 상실의 고통
- **에너지 상태**: 공포와 집착
- **자기 인식**: "나는 죽어선 안 된다"

#### 신화적 대응

> **엔키두의 죽음 후의 길가메시**
>
> 엔키두가 죽자 길가메시는 무너집니다.
> "나도 저렇게 될 것인가?"
> 그는 죽음이 두려워 광야로 도망칩니다.
> 불멸을 찾아, 죽음에서 도망쳐.
>
> **Lv1의 길가메시는 죽음 공포에 사로잡힌 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 죽음 공포 | 모든 행동의 동기 |
| 도피 | 현실을 피해 떠남 |
| 집착 | 불가능한 것을 추구 |
| 상실 부정 | 엔키두의 죽음을 받아들이지 못함 |

#### 싱크로 신호 (이런 모습이면 Lv1)

- [ ] 두려움 때문에 떠남
- [ ] 상실을 받아들이지 못함
- [ ] 불가능한 것을 추구
- [ ] 현실을 피함
- [ ] "이것만 얻으면 괜찮아질 거야"

---

### Lv2: 각성 (Awakening) — "불멸을 찾아 떠난 자"

#### 특징

```
"어딘가에 답이 있다. 찾아야 한다."
```

- **행동 패턴**: 목적 있는 탐색
- **대인 관계**: 조언자를 만남
- **에너지 상태**: 방향 있는 추진력
- **자기 인식**: "나는 탐구자다"

#### 신화적 대응

> **우트나피쉬팀을 찾아가는 길가메시**
>
> 길가메시는 대홍수를 살아남아 불멸을 얻은
> 우트나피쉬팀을 찾아갑니다.
> 세상 끝까지, 죽음의 바다를 건너.
> 목적이 있습니다 — 불멸의 비밀.
>
> **Lv2의 길가메시는 목적을 가지고 탐험하는 상태입니다.**

#### 싱크로 신호 (이런 모습이면 Lv2)

- [ ] 찾는 것이 명확함
- [ ] 조언자를 만남
- [ ] 어려움을 견딤
- [ ] 목적이 동기
- [ ] 하지만 여전히 집착

---

### Lv3: 통합 (Integration) — "빈손으로 돌아온 왕"

#### 특징

```
"불멸은 없다. 하지만 나는 다른 것을 얻었다."
```

- **행동 패턴**: 수용과 귀환
- **대인 관계**: 백성에게 돌아감
- **에너지 상태**: 평화로운 수용
- **자기 인식**: "나는 필멸자, 그리고 왕"

#### 신화적 대응

> **불사의 풀을 잃은 길가메시**
>
> 길가메시는 불사의 풀을 얻습니다.
> 하지만 뱀이 그것을 훔쳐 먹습니다.
> 그는 빈손으로 우루크에 돌아옵니다.
> 
> 하지만 돌아와 우루크의 성벽을 보며 깨닫습니다:
> "내가 쌓은 이 성벽, 이것이 나의 불멸이다."
>
> **Lv3의 길가메시는 필멸을 받아들이고 의미를 찾은 상태입니다.**

#### 싱크로 신호 (이런 모습이면 Lv3)

- [ ] 찾던 것을 얻지 못해도 괜찮음
- [ ] 여정에서 다른 것을 얻음
- [ ] 집으로 돌아옴
- [ ] 현실을 받아들임
- [ ] 남기는 것의 의미를 앎

---

### Lv4: 초월 (Transcendence) — "이야기가 된 왕"

#### 특징

```
"나의 이야기는 영원히 남는다."
```

- **행동 패턴**: 유산을 남김
- **대인 관계**: 모든 세대와 연결
- **에너지 상태**: 영원한 평화
- **자기 인식**: "나는 이야기다"

#### 신화적 대응

> **서사시가 된 길가메시**
>
> 길가메시의 이야기는 5000년이 지난 지금도 전해집니다.
> 그는 죽었지만, 그의 여정은 영원합니다.
> 인류 최초의 영웅 서사시.
> 불멸의 풀은 필요 없었습니다.
>
> **Lv4의 길가메시는 이야기로서 불멸을 얻은 상태입니다.**

## 신화 인물 3: 삼장법사 (玄奘)

| **역할** | 구법승, 서역으로 경전을 가지러 간 스님 |
| **상징** | 가사, 석장, 경전 |

### 동기 프로파일

```
Adventure:   0.70  ██████████████ (핵심)
Mastery:     0.85  █████████████████
Connection:  0.60  ████████████

```

### 핵심 특징 3가지

1. **신앙의 여정** — 진리를 찾아 떠남
2. **81난(難)** — 수많은 고난 극복
3. **가져오는 여행** — 빈손으로 가서 경전을 가져옴

---

### Lv1: 그림자 (Shadow) — "도피하는 순례자"

#### 특징

```
"여기서는 진리를 찾을 수 없다. 떠나야 한다."
```

- **행동 패턴**: 현실 불만족, 도피적 출발
- **대인 관계**: 현재의 관계 단절
- **에너지 상태**: 불안, 초조
- **자기 인식**: "여기는 내가 있을 곳이 아니다"

#### 신화적 대응

> **당나라를 떠나는 삼장**
>
