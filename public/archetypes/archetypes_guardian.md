## 원형 개요

### 핵심 정체성

**The Guardian (수호자)**는 **안정과 보호**에서 존재 의미를 찾는 원형입니다.

```
"내가 지키지 않으면 모든 것이 무너진다."
```

### 동기 가중치

| 동기 | 가중치 | 역할 |
|------|--------|------|
| **Security** | 0.65 | 핵심 동기 |
| **Connection** | 0.20 | 보조 동기 |
| **Achievement** | 0.15 | 보조 동기 |

### 원형 일치 조건

```typescript
conditions: {
  security >= 65,
  security > adventure,
  connection >= 40
}
```

### 4단계 성숙도 구조

| 레벨 | 이름 | 핵심 | 신화적 단계 |
|------|------|------|-------------|
| **Lv1** | 그림자 (Shadow) | 과잉 통제, 경직, 변화 공포 | 감옥의 수호자 |
| **Lv2** | 각성 (Awakening) | 보호의 목적, 경계 인식 | 문지기 |
| **Lv3** | 통합 (Integration) | 유연한 수호, 변화 수용 | 현명한 파수꾼 |
| **Lv4** | 초월 (Transcendence) | 존재 자체가 보호, 영원한 수호 | 신성한 문턱 |

| **역할** | 비프로스트의 수호자, 신들의 파수꾼 |
| **상징** | 갈라르 뿔피리, 황금 이빨, 비프로스트 다리 |

### 동기 프로파일


### 핵심 특징 3가지

1. **영원한 경계** — 잠들지 않고 아스가르드를 지킴
2. **초감각** — 풀 자라는 소리도 듣는 청력, 멀리 보는 시력
3. **최후의 경고** — 라그나로크 때 갈라르 뿔피리를 분다

### Lv1: 그림자 (Shadow) — "편집증적 감시자"

#### 특징

```
"모든 것이 위협이다. 절대 방심하면 안 된다."
```

- **행동 패턴**: 과잉 경계, 의심, 통제
- **대인 관계**: 모든 것을 위협으로 인식
- **에너지 상태**: 긴장, 불안, 소진
- **자기 인식**: "경계를 늦추면 모든 것이 무너진다"

#### 신화적 대응

> **잠들지 못하는 헤임달**
>
> 헤임달은 새보다 적게 잠을 잡니다.
> 밤낮으로 비프로스트를 지켜봅니다.
> 모든 소리를 듣고, 모든 것을 봅니다.
> 하지만 Lv1에서 이것은 저주가 됩니다.
> 쉴 수 없고, 믿을 수 없고, 혼자입니다.
>
> **Lv1의 헤임달은 경계에 갇혀 지쳐가는 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 과잉 경계 | 쉬지 못함 |
| 편집증 | 모든 것을 의심 |
| 고립 | 믿을 수 없어 혼자 |
| 소진 | 지속 불가능한 긴장 |

#### 싱크로 신호 (이런 모습이면 Lv1)

- [ ] 항상 긴장되어 있음
- [ ] 사람들을 믿지 못함
- [ ] 쉬면 불안함
- [ ] 모든 것이 위협으로 보임
- [ ] 통제하지 않으면 불안

---

### Lv2: 각성 (Awakening) — "비프로스트의 파수꾼"

#### 특징

```
"나는 무엇을 지키는지 안다. 그것을 위해 깨어있다."
```

- **행동 패턴**: 목적 있는 경계, 역할 인식
- **대인 관계**: 지키는 대상과 연결
- **에너지 상태**: 목표 있을 때 지속
- **자기 인식**: "나는 파수꾼이다"

#### 신화적 대응

> **아스가르드를 지키는 헤임달**
>
> 헤임달의 역할은 명확합니다.
> 아스가르드로 통하는 유일한 길, 비프로스트.
> 그곳을 지키는 것이 그의 존재 이유입니다.
> 무엇을 지키는지 알기에 견딜 수 있습니다.
>
> **Lv2의 헤임달은 자신의 역할과 목적을 아는 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 역할 고착 | 역할 외의 것을 못 봄 |
| 유연성 부족 | 상황 변화에 적응 어려움 |
| 자기 돌봄 부족 | 역할만, 자신은 없음 |
| 의미 의존 | 역할 없으면 존재감 없음 |

#### 싱크로 신호 (이런 모습이면 Lv2)

- [ ] 무엇을 지키는지 명확히 앎
- [ ] 역할에서 의미를 찾음
- [ ] 목적이 있어 버틸 수 있음
- [ ] 하지만 역할 외에는 정체성 없음
- [ ] 쉬는 것이 여전히 어려움

---

### Lv3: 통합 (Integration) — "현명한 수호자"

#### 특징

```
"지키되, 나도 지킨다. 변화도 받아들인다."
```

- **행동 패턴**: 유연한 경계, 자기 돌봄 포함
- **대인 관계**: 신뢰와 위임
- **에너지 상태**: 지속 가능한 수호
- **자기 인식**: "나는 수호자이자 나 자신"

#### 신화적 대응

> **신들과 함께하는 헤임달**
>
> 헤임달은 혼자만 경계하지 않습니다.
> 신들과 소통하고, 위협을 알립니다.
> 필요할 때는 도움을 받습니다.
> 수호는 혼자 하는 것이 아님을 압니다.
>
> **Lv3의 헤임달은 다른 이들과 함께 지키는 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 위임 어려움 | 여전히 혼자 하려 함 |
| 책임감 과잉 | 모든 것이 내 책임 |
| 변화 불안 | 변화가 여전히 불편 |

#### 싱크로 신호 (이런 모습이면 Lv3)

- [ ] 다른 이에게 맡길 수 있음
- [ ] 자신도 돌봄
- [ ] 변화에 유연하게 대응
- [ ] 긴장을 풀 수 있음
- [ ] 신뢰하는 관계가 있음

---

### Lv4: 초월 (Transcendence) — "갈라르 뿔피리의 주인"

#### 특징

```
"내가 지키는 것은 영원하다. 나는 그 문턱에 서있다."
```

- **행동 패턴**: 영원한 수호, 초월적 경계
- **대인 관계**: 모든 존재의 보호자
- **에너지 상태**: 고요한 경계
- **자기 인식**: "나는 문턱 그 자체"

#### 신화적 대응

> **라그나로크의 뿔피리**
>
> 헤임달은 라그나로크가 오면 갈라르 뿔피리를 붑니다.
> 그 소리는 아홉 세계에 울려 퍼집니다.
> 최후의 전투를 알리는 소리.
> 그는 로키와 싸워 함께 죽습니다.
> 
> 하지만 그의 수호는 끝나지 않습니다.
> 새 세계가 시작되어도 누군가 문턱을 지킬 것입니다.
>
> **Lv4의 헤임달은 영원한 수호의 원형이 된 상태입니다.**

#### 도달 경지

| 경지 | 발현 |
|------|------|
| 영원한 경계 | 죽어서도 수호 |
| 최후의 경고 | 필요할 때 뿔피리 |
| 문턱 자체 | 존재가 경계 |
| 평화로운 경계 | 긴장 없이 깨어있음 |

#### 싱크로 신호 (이런 모습이면 Lv4)

- [ ] 경계하되 긴장하지 않음
- [ ] 존재 자체가 보호
- [ ] 언제 경고해야 할지 앎
- [ ] 끝까지 역할을 다함
- [ ] 다음 세대에게 넘김

---

### 레벨 진화 조건

```
Lv1 → Lv2: 무엇을 왜 지키는지 명확히 알 때
Lv2 → Lv3: 혼자가 아닌 함께 지킬 때
Lv3 → Lv4: 존재 자체가 수호가 될 때
```

### 레벨 퇴화 경고

```
Lv4 → Lv3: 다시 모든 것을 혼자 하려 할 때
Lv3 → Lv2: 신뢰가 무너지고 고립될 때
Lv2 → Lv1: 목적을 잃고 편집증에 빠질 때
```

### Lv1: 그림자 (Shadow) — "껍데기에 갇힌 자"

#### 특징

```
"밖은 위험하다. 절대 나가지 않겠다."
```

- **행동 패턴**: 회피, 은둔, 방어 과잉
- **대인 관계**: 단절, 고립
- **에너지 상태**: 정체, 움직이지 않음
- **자기 인식**: "껍데기 안이 안전하다"

#### 신화적 대응

> **움직이지 않는 거북**
>
> 거북이는 위험을 감지하면 껍데기 안으로 들어갑니다.
> 안전하지만, 아무것도 할 수 없습니다.
> Lv1의 현무는 껍데기에서 나오지 않습니다.
> 지키는 것이 아니라, 숨는 것입니다.
>
> **Lv1의 현무는 보호가 아닌 회피 상태입니다.**

#### 전형적 함정

| 함정 | 발현 |
|------|------|
| 회피 | 문제를 피함 |
| 고립 | 세상과 단절 |
| 정체 | 변화를 거부 |
| 무기력 | 아무것도 안 함 |

#### 싱크로 신호 (이런 모습이면 Lv1)

- [ ] 새로운 것을 피함
- [ ] 안전지대에서 나오지 않음
- [ ] 변화가 두려움
- [ ] 사람들과 단절
- [ ] "가만히 있으면 안전해"

---

### Lv2: 각성 (Awakening) — "북방을 지키는 자"

#### 특징

```
"나는 북쪽을 지킨다. 그것이 나의 역할이다."
```

- **행동 패턴**: 자리를 지킴, 역할 인식
- **대인 관계**: 다른 사신들과 협력
- **에너지 상태**: 안정적, 느리지만 꾸준함
- **자기 인식**: "나는 북방의 수호자"

#### 신화적 대응

> **사신 중 하나인 현무**
>
> 현무는 청룡, 백호, 주작과 함께 사신입니다.
> 각자의 방위를 지킵니다.
> 현무는 가장 어둡고 추운 북쪽.
> 아무도 원하지 않는 곳을 묵묵히 지킵니다.
>
> **Lv2의 현무는 자신의 자리를 알고 지키는 상태입니다.**

#### 싱크로 신호 (이런 모습이면 Lv2)

- [ ] 자신의 역할을 앎
- [ ] 묵묵히 자리를 지킴
- [ ] 다른 이들과 협력
- [ ] 어려운 곳도 맡음
- [ ] 하지만 변화는 여전히 불편

---

### Lv3: 통합 (Integration) — "거북과 뱀의 조화"

#### 특징

```
"지키되 깨어있다. 방어와 경계의 균형."
```

- **행동 패턴**: 방어(거북)와 경계(뱀)의 통합
- **대인 관계**: 적극적 보호
- **에너지 상태**: 고요하지만 준비됨
- **자기 인식**: "나는 완전한 수호자"

#### 신화적 대응

> **거북과 뱀의 결합**
>
> 현무는 거북이 몸에 뱀이 감긴 형상입니다.
> 거북은 방어 — 단단한 껍데기.
> 뱀은 경계 — 날카로운 감각.
> 둘이 하나가 될 때 완전한 수호가 됩니다.
>
> **Lv3의 현무는 방어와 경계를 함께 가진 상태입니다.**

#### 싱크로 신호 (이런 모습이면 Lv3)

- [ ] 수비만이 아닌 적극적 보호
- [ ] 위협을 미리 감지
- [ ] 필요할 때 나설 수 있음
- [ ] 고요하지만 깨어있음
- [ ] 변화에 유연함

### Lv4: 초월 (Transcendence) — "북극성의 수호자"

#### 특징

```
"나는 북쪽 그 자체다. 변하지 않는 기준점."
```

- **대인 관계**: 모든 이의 기준점

- **자기 인식**: "나는 북극성처럼 변하지 않는다"

>
> **Lv4의 현무는 변하지 않는 기준점이 된 상태입니다.**

### Lv1: 그림자 (Shadow) — "죽음에 집착하는 자"

#### 특징

```
"죽음만이 확실하다. 모든 것은 사라진다."
```

- **행동 패턴**: 비관, 죽음 집착
- **대인 관계**: 삶을 불신
- **에너지 상태**: 어둠, 무거움
- **자기 인식**: "나는 죽음의 신"

#### 신화적 대응

> **죽은 자들 사이의 아누비스**
>
> 아누비스는 죽은 자들과 함께합니다.

- **에너지 상태**: 안정적
- **자기 인식**: "우리는 사천왕이다"

#### 신화적 대응

> **각자의 방위를 지키는 사천왕**
>
> 지국천왕은 동쪽, 증장천왕은 남쪽,
> 광목천왕은 서쪽, 다문천왕은 북쪽.
> 각자의 자리에서 최선을 다합니다.
>
> **Lv2의 사천왕은 각자의 역할을 하는 상태입니다.**

---

### Lv3: 통합 (Integration) — "함께 지키는 자들"

#### 특징

```
"우리는 하나다. 함께 지킨다."
```

- **행동 패턴**: 완전한 협력
- **대인 관계**: 진정한 팀
- **에너지 상태**: 시너지
- **자기 인식**: "함께할 때 완전하다"

#### 신화적 대응

> **불법을 함께 수호하는 사천왕**
>
> 사천왕은 함께 불법을 수호합니다.
> 하나가 약해지면 다른 셋이 돕습니다.
> 빈틈이 없습니다.
> 넷이 하나가 될 때 완전한 수호가 됩니다.
>
> **Lv3의 사천왕은 완전한 협력을 이룬 상태입니다.**

---

### Lv4: 초월 (Transcendence) — "사방이 곧 중심"

#### 특징

```
"어디든 중심이다. 모든 곳을 지킨다."
```

- **행동 패턴**: 존재가 수호
- **대인 관계**: 모든 존재와 연결
- **에너지 상태**: 편재하는 보호
- **자기 인식**: "우리는 수호 그 자체"

#### 신화적 대응

> **모든 사찰의 입구에 선 사천왕**
>
> 사천왕은 모든 불교 사찰의 입구에 있습니다.
> 천왕문(天王門)을 지킵니다.
> 들어가는 모든 이를 보호하고,
> 악을 막습니다.
>
> **Lv4의 사천왕은 보편적 수호의 상징입니다.**
